# 相關成員

相關成員是專案的外部人員，用來記錄與公司密切關聯的利害關係人，包括**業主**、**起造人**、**監造人**、**承造人**、**營造人**、**設計師**、**建築師**、**專業技師**、**建管單位**。



## 新增相關成員

進入 「 相關成員 」 的介面後，點選右上角 「 + 新增成員 」，輸入手機號碼搜尋特定人員，選擇該人員後即可設定 「 角色性質 ( 可複選 ) 」 並增添備註。


!!! info
    該成員須先以手機號碼註冊 Jobdone 帳號。
