# 廠商 > 合約格式

## 協力廠商


!!! info
    若尚未設定協力廠商，請參閱[**協力廠商設定**](../../contractor#xin-zeng-xie-li-chang-shang)。


若選擇廠商 > 合約格式，系統會自動顯示協力廠商列表，可點選列表中的廠商編輯合約或材料項目。

若材料項目**尚未**指派或發包給協力廠商，可將施工項目建立於 **「 (未指派) 」** 分類中。


!!! warning
    若材料項目已指派廠商，就無法變更為其它的廠商


![](../../images/7c41d37a97e83ca1.png)

## 合約 <a href="#contract" id="contract"></a>

點選協力廠商後會顯示合約清單，點選右上角 「 編輯 」 ，即可新增或刪除合約。

![](../../images/3cdc3474dc7277a8.png)

![](../../images/b14e83823162c826.png)

## 材料項目 <a href="#construction_item" id="construction_item"></a>

### 設定材料分類 <a href="#categories" id="categories"></a>

點選右上角 「 新增 」 ，先選擇分類側邊的 「 編輯 」 ，即可新增或刪除分類，完成後點選 「 新增 」 儲存。

![](../../images/7e651982b639058a.png)

### 新增材料項目 <a href="#create" id="create"></a>

點選右上角 「 新增 」 ，先選擇材料項目所屬的合約，即可開始填寫材料項目內容，完成後點選 「 新增 」。


!!! info
    若未選擇合約，施工項目將歸類到 「 沒有合約 」 分類中。


![](../../images/6b0cd30f81f41879.png)

### 編輯 / 移動 / 刪除材料項目 <a href="#edit" id="edit"></a>

點選 「 編輯本頁 」 按鈕即可**編輯**材料項目內容，也可以點選 「 **⋮** 」 將材料項目**刪除**或**移動至其它合約**，編輯完成後點選 「 儲存 」 。

![](../../images/1d57341478da1b77.png)

### 批次設定協力廠商 / 合約 <a href="#file_import" id="file_import"></a>

點選 「 編輯本頁 」 按鈕後，使用列表前方的核取方格勾選多筆材料項目，然後點選 「 設定協力廠商 」，即可將多筆材料項目移至選定的協力廠商或合約項目下。

![](../../images/11a8af5bbdd41834.png)

## 從檔案匯入 <a href="#file_import" id="file_import"></a>

材料項目可使用指定格式的 Excel 批次匯入，點擊 「 Excel 匯入 」 按鈕開啟檔案匯入功能。


!!! warning
    檔案匯入功能僅可以在沒有任何合約及施工項目的情況下使用。


### 下載並匯入 Excel 模板 <a href="#file_download" id="file_download"></a>

點選右上角的 「 Excel匯入 」，下載 「 材料管理 Excel 模版 」，並使用模板填妥資料。上傳檔案後點選 「 匯入 」 即可批次匯入施工項目資料。

![](../../images/85144e3e06a234b6.png)
