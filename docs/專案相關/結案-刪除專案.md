# 結案 / 刪除專案

## 結案 / 刪除專案

點選專案右方的 「 **⋮** 」 按鈕，即可將專案 「 設為已結案 」 或 「 刪除 」。


!!! warning
    請注意：結案的專案**無法改回**進行中，專案刪除後也將**無法復原**，請謹慎操作。


![](images/7d3539322087afa8.png)

![再次輸入專案名稱並選擇 「 刪除專案 」](images/12920c29562744a4.png)

## 查看已結案清單

點選 「 已結案 」 分頁，可查看所有已列為結案之專案。
