# 協力廠商 / 外部聯絡人

協力廠商 / 外部聯絡人用來設定和本專案有往來的廠商 / 個人相關資料，可在後續的功能中選取。


!!! warning
    協力廠商／外部聯絡人須先註冊 Jobdone 帳號 (無需付費)。


***

## 協力廠商

### 新增協力廠商

進入 「 協力廠商 / 外部聯絡人 」 介面後，點選右上角 「 + 協力廠商 」，輸入公司名稱、統一編號或開業證號，即可新增協力廠商。

若協力廠商**未註冊** Jobdone 帳號，可點選 「 新增自訂廠商 」 直接輸入公司名稱，但無法使用關聯功能。

![可輸入公司資訊搜尋協力廠商，也可以選擇 「 新增自訂廠商 」。](images/b8ce4fc74d2b9170.png)

## 外部聯絡人

### 新增外部聯絡人

進入 「 協力廠商 / 外部聯絡人 」 介面後，選擇 「 外部聯絡人 」 分頁，點選右上角 「 + 外部聯絡人 」，輸入手機號碼後即可新增外部聯絡人。


!!! info
    若您新增的外部聯絡人帳號已在某家公司，此公司也會同時加入你的協力廠商。
