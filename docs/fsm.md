# 現場服務 FSM\*\*

#### 現場服務也是工程承攬或服務的一部份，與營建不同的是現場服務可能包含了臨時性、合約周期性的行為，大多是以"客戶—案件"為單位，也不像營建的專案有明確的整個專案結案目標，會在建造完成後結束。這類的行為應該是最廣泛的，例如：交屋後的保固；電梯、消防、機電或其它設備的定檢維護；開口合約的派工，像道路、監控、交通設施等；這裏又包含了派工的對象，可能是內部或者外部的人員，複雜程度不一定就比營造來的低。
