# 新增 / 編輯臨時工

## 01｜新增派遣人力

{% stepper %}
{% step %}
### 新增派遣工

如下圖紅框圈選處，進入臨時工管理頁面後，點選右上角&#x4E4B;**「＋新增派遣人力」**，即可開始輸入臨時工電話並驗證。

![新增派遣人力](../../../images/0a543a1122e808da.png)
{% endstep %}

{% step %}
### 填寫臨時工電話

將臨時工電話填寫完畢並經過驗證，便可開始填寫其詳細資料。


!!! warning
    由於需要電話驗證，務必確認欲新增之派遣工已&#x65BC;**「臨時工接單 App」**&#x8A3B;冊帳號。


![輸入臨時工電話](../../../images/0f0826b1fbf61675.png)

將電話填寫完畢後，即可點選右方&#x4E4B;**「驗證」**。

若驗證成功則可開始填寫詳細資料；若驗證失敗則會跳出**找不到資料**之提示。

![](../../../images/2830de9fbc95a3cf.png)

以下為**驗證失敗**與**驗證成功**之範例。


!!! warning
    若驗證失敗，請您再次確認輸入之號碼是否有誤，或聯絡臨時工儘快註冊帳號。


<div>![驗證成功](../../../images/b138cf4ce28d7301.png) ![驗證失敗](../../../images/160a7b9969d1091d.png)</div>
{% endstep %}

{% step %}
### 填寫臨時工資料

詳細資料如下，請您妥善填寫臨時工之個人檔案。

<div>![臨時工資料填寫(一)](../../../images/85d59dd39d76991c.png) ![臨時工資料填寫(二)](../../../images/c8e94b9c23d17aa8.png)</div>

## 02｜編輯臨時工

如左圖紅框圈選處，於欲編輯之派遣工右側**操作**欄位，點&#x9078;**「編輯臨時工」**，即可如右圖開始編輯其資料。

<div>![左圖-點選「編輯臨時工」](../../../images/7080a12bbb2acf6d.png) ![右圖-編輯臨時工資料](../../../images/432d9f6aecac04ea.png)</div>

***

## 03｜刪除臨時工

如左圖紅框圈選處，於欲刪除臨時工右側**操作**欄位，點&#x9078;**「刪除臨時工」**。

<div>![左圖-點選「刪除臨時工」](../../../images/29373fa28d7c53a7.png) ![右圖-確定刪除/取消](../../../images/bf283804221e513f.png)</div>
