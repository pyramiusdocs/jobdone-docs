# 人力派遣 TWA

此功能專為仲介派遣公司設計，旨在解決與客戶點工需求相關的各項作業問題。

包括：叫工、派工、報到、簽退、計價等流程，減少傳統人工作業的複雜性，提升派遣管理效率。


!!! tip
    企業能夠更高效地管理派遣人力，降低溝通成本，確保每個用工環節皆能即時追蹤並準確記錄


***

Jobdone 的人力派遣模組專為『營造工程點工』模式量身打造。無論是應對工地臨時性的用工需求、確保工力及時到位，或是管理自有工班的每日出勤，本模組皆能取代傳統繁瑣的電話叫工與紙本簽單，改以數位化方式精準作業。

{% tabs %}
{% tab title="臨時點工需求" %}
針對非承攬性質、需按日計酬的零星工項，快速對接外部人力資源。
{% endtab %}

{% tab title="自有工班管理" %}
適用於以『每日出工』為作業單位的團隊，詳實紀綠每日出勤人數與工作內容，而非單純以工項進度作為結算依據。
{% endtab %}
{% endtabs %}

**三大角色身份與權責 (The Three Pillars)**

為了實現高效的媒合與管理，系統將使用者依身份差異區分為以下三大類別：

{% tabs %}
{% tab title="A. 營建商 (Client)" %}
* 角色功能： 需求發起者。
* 關鍵操作： 於系統內發布點工需求（包含工種、人數、工作地點、預計工時等），並針對到場的派遣工執行線上簽到確認與評價，確保出工品質與合約相符。
{% endtab %}

{% tab title="B. 派遣商 (Dispatch Agency)" %}
* 角色功能： 資源供應者（人力仲介或專業分包商）。
* 關鍵操作： 接收營建商的需求派單，並將任務精準指派給旗下的派遣工。透過系統即時掌握人力動向，並作為後續請款與發放薪資的數位憑證。
{% endtab %}

{% tab title="C. 派遣工 (Dispatch Laborer)" %}
* 角色功能：現場執行者。
* 關鍵操作：透過 App 接收工作通知，並於抵達工地執行 GPS 定位簽到。任務完成後可即時回報工作照片，確保當日工時紀錄準確無誤，保障自身領薪權益。
{% endtab %}
{% endtabs %}
