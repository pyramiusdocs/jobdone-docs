# 專案經理 / 專案成員

## 權限介紹

### 專案經理

* 可編輯專案的所有資訊
* 將成員加入專案
* 授予專案經理權限
* 將成員請離專案

### 專案成員

* 查看專案資料，但僅能變更部分資料
* 可使用專案所有功能

***

## 網頁版

### 將成員加入 / 請離專案

點選左側選單 「 專案成員 」 進入頁面後，點選右上角 「 管理成員 」，即可將指定公司成員加入或移出專案，並可授權是否為專案經理。

![進入 「 專案成員 」 頁面後，點選 「 管理成員 」](../assets/images/6d0d6df91ad8e5c8.png)

![將指定成員加入或移除專案，並可設定是否為專案經理](../assets/images/e91cbb4ce7590bfd.png)

### 修改職稱

點選專案成員右方的 「 **⋮** 」 按鈕，選擇 「 查看詳細資訊 」，即可選擇設定好的[**專案職稱**](../company_level/commonsetting)。

![](../assets/images/b1686a7ac34bee45.png)

![](../assets/images/5ec2e69d844f247f.png)
