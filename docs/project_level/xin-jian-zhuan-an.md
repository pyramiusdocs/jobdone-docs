# 新建專案

## 新建專案


點選右上角的新建專案，依序選擇[功能模組]() ( 目前包含[一般建案]()和[公共工程]() )、輸入專案基本資料、選擇專案身份 ( 營造 / 監造 )，並指派負責的專案經理，即可完成建立新專案。
