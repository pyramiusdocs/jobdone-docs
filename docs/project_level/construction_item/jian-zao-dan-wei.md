# 監造單位

若您是監造單位，設定施工項目須先選擇 「 自行建立施工項目 」 或 「 關聯營造專案 」。


# 自行建立施工項目


若點選 「 自行建立施工項目 」，可參考 [施工項目]() 。新增 / 刪除 / 移動 / 編輯施工項目


![](../../assets/images/Kc2OxdXeUBJMtxmOPWlV.png)


# 關聯營造專案


為方便監造單位進行工程管理，可使用關聯功能一鍵複製營造單位建立好的工項。



## 新增關聯專案


若您尚未關聯營造專案，請點選 「 前往關聯營造專案，稍後返回此頁複製 」。


!!! info
    關聯方法請參照 [與監造/營造單位專案關聯]()


![](../../assets/images/230q0jMPUQelvwKXSWOk.png)


## 複製施工項目


**關聯成功後**，進入施工項目點選 「 從營造專案複製施工項目及總表」 按鈕，即可一鍵複製施工項目。


!!! warning
    關聯僅能複製**現有施工項目**，若未來營造單位若有編輯或更動工項，仍然需要手動更新。
