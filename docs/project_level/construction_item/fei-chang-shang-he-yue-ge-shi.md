
# 施工項目


## 新增施工項目


點選右上角 「 新增 」 ，開始填寫施工項目表單，完成後點選 「 新增 」。


![](../../assets/images/8O4Nky1Tgv34ErYEDPMN.png)


![](../../assets/images/7PYMAHSpsd7hiaMIovfp.png)


## 編輯 / 移動 / 刪除施工項目


點選 「 編輯本頁資料 」 按鈕即可**編輯**施工項目內容，也可以點選 「** ⋮** 」 將施工項目**刪除**，編輯完成後點選 「 儲存 」 。


![](../../assets/images/rTtJGGlWJjYGlz89F89O.png)





# 從檔案匯入


施工項目可使用指定格式的 Excel 批次匯入，點擊 「 Excel 匯入 」 按鈕開啟檔案匯入功能。


!!! warning
    檔案匯入功能僅可以在沒有施工項目的情況下使用。


![](../../assets/images/MIGX8J7jtXlllfUIYckJ.png)


## 下載並匯入 Excel 模板


點選右上角的 「 Excel匯入 」，下載 「 施工項目 Excel 模版 」，並使用模板填妥資料。上傳檔案後點選 「 匯入 」 即可批次匯入施工項目資料。


![](../../assets/images/afLjJz6fhVURRwbnF2rv.png)
