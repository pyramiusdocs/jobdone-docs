
# 結案 / 刪除專案


點選專案右方的 「** ⋮** 」 按鈕，即可將專案 「 設為已結案 」 或 「 刪除 」。


!!! warning
    請注意：結案的專案無法改回進行中，專案刪除後也將無法復原，請謹慎操作。

![](../assets/images/Fjv4IbMowDwOQvMLlUlQ.png)


# 查看已結案清單


點選 「 已結案 」 分頁，可查看所有已列為結案之專案。
