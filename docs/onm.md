# 場域維運 O&M**

Operation & Maintenance

### 場域維運很多是EPC的公司，管理的是固定範圍的場域以及設備的維運，這個場域可能是自有的、BOT、或簽約期間代管。無論如何從EPC的角度，對上可能會有業主、對下會有不同性質的承攬商。工作的內容可能包含定巡檢、緊急維修。新一點的案場重要的設備會整合IoT或SCADA。還有工項相對應的人員進場申請管理、證照、保險等。相關的功能概念可以參見https://blog.jobdone.cc/series/maintenance
