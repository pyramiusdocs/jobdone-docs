# APP 版

## 存入相簿設定

登入App後，點選左上角 「 設定 」 圖示，點選 「 裝置設定 」，即可設定是否將照片存入相簿中。

## 工程相機（帶專案浮水印）

其內部資訊包括：**「拍攝時間」**、**「專案名稱」**、**「地點」**、**「GPS」**、**「拍攝者」**、**「拍攝者公司」**、**「備註」**&#x7B49;。
