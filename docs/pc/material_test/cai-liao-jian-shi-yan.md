# 材料檢試驗

送審資料

!!! info
    建立材料設備管制總表總表前，需先完成設定[材料管理]()。


# 建立材料設備管制總表

1. 進入專案介面，選擇 「 材料檢試驗 」 進入管理介面。
2. 點選左上角 「 選擇材料 」 
3. 根據材料篩選器中所設的篩選條件，勾選欲加入總表的材料項目。


![](../../assets/images/YYBHAk5wSwIGZp7NWpmo.png)


![](../../assets/images/CXMU6VteJTOgEs0CtfQA.png)


![](../../assets/images/7m7RlcoOfhUAU66exN2Q.png)


# 申請檢驗

1. 點選材料右側的編輯按鈕。
2. 填寫完檢驗資訊以後，點選儲存，即可完成申請檢驗。


![](../../assets/images/ShkDBSLX1fJQxkRiyAww.png)


![](../../assets/images/kB9OFnZHvPPQS2rH7rnQ.png)


# 功能列（篩選、檢視、搜尋）

- 「 **篩選 **」 功能可指定搜尋條件，快速查找所需要的材料項目。
- 「 **檢視方式 **」，可切換 「 送審管制表 」 與 「 檢試驗管制表 」 兩種瀏覽模式，也可以選擇 「 顯示全部 」。
- 可以透過 「 **搜尋 **」 輸入簡易的材料名稱關鍵字，快速查找所需要的材料項目。


![](../../assets/images/yALPntiaUleNhhZi6Uqt.png)


# 品質管理標準

1. 點擊右上角的 「 品質管理標準 」 按鈕。
2. 點選 「 上傳檔案 」 即可上傳已編寫好的品管標準表。


!!! info
    建議上傳檔案以PDF檔案為主。
