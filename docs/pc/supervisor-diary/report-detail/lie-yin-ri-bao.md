# 列印日報

## 列印日誌


!!! info
    為確保列印品質，建議使用 Google Chrome 瀏覽器列印。

1. 點選日報頁面的右上角的 「 列印預覽 **」 **按鈕
2. 確認預覽內容無誤後，點選右下角的 「 列印報告** 」** 按鈕，即可進行列印。


!!! warning
    在[列印監造日報]()時，**不會**將其他應檢附文件印出，須自行另外列印。
