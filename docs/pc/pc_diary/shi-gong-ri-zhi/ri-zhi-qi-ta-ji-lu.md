# 日誌 / 其他紀錄

在當日的日誌右上角處，提供各式其他紀錄事項進行填寫。


![](../../../assets/images/1T0Wlj6ERf81gxA0Xnfo.png)





# 填寫其他事項紀錄


點選 「 ＋其他事項紀錄 」，填寫內容後選擇 「 新增紀錄 」，即可完成。


!!! info
    若要新增其他日期的紀錄，或是跨日期檢視其他紀錄，可從[其他事項紀錄總表]()進行操作。


![](../../../assets/images/1VputCuYTIryu0YOjId2.png)


![](../../../assets/images/zrbylyZU0xKSw7P0EWtO.png)
