# 日誌 / 施工概況

紀錄當日施工項目 、 完成數量 、 備註 的相關資訊。

!!! warning
    填寫日誌其他內容之前，必須先填寫[基本資訊]()。


# 編輯施工概況

1. 進入施工日誌詳情後，點選 「 施工概況 」。
2. 選擇協力公司後，用篩選器根據條件選擇工項（條件設定後要按一下放大鏡按鈕！）
3. 填寫 「 本日完成 」 欄位及備註，即可儲存變更。


!!! info
    若尚未設定工項，請先至專案介面設定[施工項目]()。

!!! info
    ### 營造業專業工程特定施工項目
    
    
    選擇的工項當中，若需要專業技術士的工項就會自動被分配至**營造業專業工程特定施工項目**。
    
    
    而工項是否需要專業技術士，可在[新增施工項目]()中進行設定。
