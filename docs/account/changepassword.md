# 修改密碼

Change Password

---


!!! warning
    請注意：密碼僅能在網頁版中進行修改。





### 一、進入個人資訊頁面，點選 「 修改密碼 」 。


![](../assets/images/0DCv6pFmPd1rY0fmntXT.png)


![](../assets/images/8a2rjN9Va6PO0FhuXFib.png)


### 二、輸入想要變更的密碼，輸入完畢後點選 「 變更密碼 」。


![](../assets/images/ANimVwRIqLxBoDxf3EIM.png)


### 三、跳出 「 密碼更新成功 」 即可。


![](../assets/images/olW9zOG59fn6h1UdgrLM.png)
