# 個人資料




!!! info
    個人資料是由帳號擁有者自行建立及更新，建議您填寫可容易辨識的名稱及頭像，讓其他人更容易找到您。




## 網頁版

#### 一、登入後，點選左下角進入個人資訊頁面。

![](../assets/images/423da585eedd7cff.png)

![](../assets/images/28447d2185780d25.png)

#### 二、點選右上角 「 編輯 」 即可修改個人資訊。

![](../assets/images/455061a3a66c19d5.png)

#### 三、修改完成後，按下右上角 「 儲存 」 即可變更資訊；按下 「 取消 」 ，則可恢復原來的資訊。

![](../assets/images/74a7a55932aba48c.png)

**請注意：如需變更手機，需重新驗證**[**手機號碼**](../sign_up#er-yan-zheng-shou-ji)。
