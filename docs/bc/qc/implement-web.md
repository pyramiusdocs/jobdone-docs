

Assign & Response to Observations
# 網頁版


## 建立檢查工作


在檢查表中選擇 「 執行檢查工作 」 頁面，然後點選 「 +檢查工作 」 ，即可新增檢查工作。


!!! info
    **欄位說明 / 備註**
    
    - **選用檢查範本**：選擇於[設定專案檢查範本]()建立的檢查範本。
    - **編號**：可於建立檢查工作時指定編號，若無指定則由系統自動產生編號。編號於檢查建立完成後無法更改。
    - **協力廠商**：於 「 [協力廠商]() 」 建立的協力廠商列表，點選下拉式選單選取。
    - **預計執行日期**：除了選擇特定日期外，亦可勾選 「 批次建立 」 選擇多個日期。
    - **檢查人**：於 「 [專案成員]() 」 建立的成員列表，點選下拉式選單選取職位及成員。
    - **檢查可能使用圖面**：於 「 [施工圖面]() 」 建立的施工圖面列表，點選 「 增加圖面 」 選取圖面。
    - **檢查位置**：於 「 [建地結構]() 」 建立的建地結構列表，使用選單選取建地結構，或手動輸入檢查位置描述。
    - **標籤**：使用標籤對檢查進行分類管理。


![](../../assets/images/mdRoi5wgzlUKpA5j8nEa.png)


## 建立周期性檢查表


若需建立周期性執行的檢查表，可於新增檢查工作處勾選 「 批次建立 」 ，選擇多個日期。


!!! info
    批次建立僅可從網頁版操作，可建立多張周期性的檢查表，例如：每天、每周、每月。


![](../../assets/images/Dlgv05BolUCIFhWRtZ94.png)


## 執行檢查


在檢查工作列表中選擇欲執行的檢查表，點選檢查項目，即可顯示該檢查的詳細紀錄。


在執行檢查工作時，可直接點選下拉式選單選取檢查結果，也可以點選右方 「 + 紀錄 」 ，填寫合格狀態、紀錄檢查標準、文字描述、圖面標示、附圖及附件等。每個檢查項目可新增多個紀錄，詳細描述檢查結果。


完成檢查後，點選 「 檢查完成，送審 」 按鈕完成此檢查。


!!! warning
    檢查工作需全部填寫完成才可以進入下一階段。

![](../../assets/images/MIrobXHTdqzbQtD0l9eH.png)


## 一鍵不適用 / 未填寫、新增自訂檢查項目


點擊下圖紅框處的 「 ⋮ 」 選單圖示，可以對檢查工作選擇 「 一鍵不適用 」 或 「 一鍵未填寫 」 ，一次更改所有檢查工作的狀態。也可以選擇 「 新增自訂檢查項目 」 ，將自訂的檢查項目加入檢查列表中。


![](../../assets/images/yq2V3EUlnmI9qOSXEa6g.png)


# APP


## 建立檢查工作


進入後檢查表後，點選右下方的 「 ＋ 」 號就可以開始新增檢查工作。填寫檢查相關資訊後，點選下方的 「 建立新的檢查 」 。


!!! info
    **欄位說明 / 備註**
    
    - **檢查範本與選用題目類別**：選擇於[設定專案檢查範本]()建立的檢查範本。
    - **協力廠商**：於 「 [協力廠商]() 」 建立的協力廠商列表，點選下拉式選單選取。
    - **檢查人**：於 「 [專案成員]() 」 建立的成員列表，點選下拉式選單選取職位及成員。
    - **檢查可能使用圖面**：於 「 [施工圖面]() 」 建立的施工圖面列表，點選 「 增加圖面 」 選取圖面。
    - **檢查位置**：於 「 [建地結構]() 」 建立的建地結構列表，使用選單選取建地結構，或手動輸入檢查位置描述。


![](../../assets/images/lapeYvDtuwPKEP7pJb5M.png)


![](../../assets/images/CddsGDZDtzU1MuW9tRp9.png)


## 執行檢查 / 新增檢查紀錄


點選檢查項目後，即可直接點選下拉式選單選取檢查結果，也可以點選項目右方 「 + 」 按鈕 ，填寫合格狀態、紀錄檢查標準、文字描述、圖面標示、附圖及附件等。每個檢查項目可新增多個紀錄，詳細描述檢查結果。


完成檢查後，點選 「 初驗完成，送交審核 」 完成檢查。


!!! info
    若手機處於離線模式，仍然可以建立檢查紀錄，並於恢復連線後點選 「 未上傳的離線紀錄 」 重新上傳檢查紀錄。


![](../../assets/images/efCoaqQZdr5WhDi5E7e2.png)


![](../../assets/images/NfwIQWyS6kFHSP4aMuFW.png)

點擊畫面 「 更多功能 ⋮ 」 圖示，可以對檢查工作選擇 「 一鍵不適用 」 或 「 一鍵未填寫 」 ，一次更改所有檢查工作的狀態。也可以選擇 「 新增自訂檢查項目 」 ，將自訂的檢查項目加入檢查列表中。
