

初驗審核結束後，檢查表狀態會變更為 「 待複驗 」 ，在列表中可點選 「 待複驗 」 按鈕篩選已完成初驗的檢查表。
# 網頁版


## 複驗


進入檢查表後，點選 「 不合格 」 的檢查結果，右方檢查紀錄會顯示 「 + 複驗紀錄 」 或 「 發起改善 」 按鈕，點擊按鈕後即可填寫複驗紀錄或發起改善單。


![](../../assets/images/4Lhu87zspUYnmnmFs1k0.png)


## 填寫複驗紀錄


點選 「 + 複驗紀錄 」 即可填寫複驗紀錄，填寫完畢後點選 「 儲存複驗結果 」。


!!! info
    在[列印檢查報告]()時，若選擇 「 交付模式 」 則會將複驗紀錄取代不合格紀錄，並標記為合格。


![](../../assets/images/ZpqpwcSpbMmIQJniBp3o.png)


## 發起改善單


針對不合格缺失，可以發起改善單追蹤改善情況，點選 「 發起改善 」 ，即可[發起改善單]()。


!!! info
    若建立不合格紀錄時有指定缺失責任人，此處會自動帶入缺失責任人作為改善單的受文者。若想選擇其他受文者也可以點選 「 重新選擇 」 變更受文者。


![](../../assets/images/8ltt5cBsEn9eMfBnX5j7.png)
