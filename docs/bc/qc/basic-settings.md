# 檢查表設定

調整檢查相關設定後，檢查流程將會隨之變動，可調整的設定包含：

## 初驗審核 <a href="#init-exam" id="init-exam"></a>

設定為啟用時，檢查流程會包含初驗審核，檢查會需要經過專案經理審核後才可進入下一階段。\
設定為停用時，執行檢查工作初驗完成後不需送審，檢查人初驗完成後即可自行結案。

## 複驗 <a href="#re" id="re"></a>

設定為啟用時，可針對不合格的檢查紀錄填寫複驗內容。\
設定為停用時，無法使用 「 複驗 」 及 「 複驗審核 」 功能，送審且審核通過後便會進入結案狀態。

## 複驗審核 <a href="#re-exam" id="re-exam"></a>

設定為啟用時，複驗完成後會進入複驗審核狀態，複驗內容填寫完成後，需經過專案經理審核才能結案。\
設定為停用時，無法使用 「 複驗審核 」 功能，複驗完成後可自行結案。

## 缺失責任人 <a href="#pic" id="pic"></a>

設定為啟用時，**可以**在發起改善單時可以快速帶入缺失責任人 ( 須填寫不合格紀錄時標記缺失責任人 )。\
設定為停用時，**無法**在發起改善單時快速帶入缺失責任人。

## 改善單發起 <a href="#improvement" id="improvement"></a>

設定為啟用時，**可以**在檢查表中發起改善單。\
設定為停用時，**無法**在檢查表中發起改善單。
