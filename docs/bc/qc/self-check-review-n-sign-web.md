# 審核檢查表

Approve & Close Observations

檢查表送審後，待審核的檢查表會顯示 「 審核中 」 狀態，擁有專案經理權限的人可對檢查表進行審核，結果包含 「 [退回修改]() 」 及 「 [通過並簽名]() 」 。


# 網頁版


## 退回修改


點選 「 退回修改 」 後，檢查表狀態將改為 「修正中」 並退回給檢查人員，允許檢查人員修正檢查表內容。


![](../../assets/images/vzKnKUiuZgVOkmLcj6uT.png)


![](../../assets/images/vy7Pb3NbPRtRhcaTySQR.png)


## 通過並簽名


當檢查表內容無誤時，可點選 「 通過並簽名 」 進入[複驗]()階段。若不需要複驗，可在 「 檢查表設定 」 中停用此功能。


!!! warning
    按下 「 通過並簽名 」 進入複驗後，檢查表就無法進行任何改動，也無法退回審核階段，在結案前請確認內容無誤。


![](../../assets/images/WbdlOiIpI6d7mq34IobW.png)


# APP


進入專案內的檢查表後，選擇 「 審核中 」 分頁即可看見待審核的檢查表。


審核完畢後點選下方的 「 審核通過 」 或 「 退回 」 即可完成審核。
