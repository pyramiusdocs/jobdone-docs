# 營建管理 CPM

在台灣，可能公共工程都習慣叫 『PMIS』。

Jobdone，是一個專為工程的產業鍊管理而生的跨公司、跨專案平台。不止服務營造公司，也滿足營造的上下游管理。

營造工程是一個很特殊的專案型態，在一個工地外部的成員會遠超過單一公司的人員，用傳統的資訊軟體觀念開發的專案管理或流程表單，只能在作為單一案場的工作紀錄，無法真正解決產業鍊上多角色參與的整理優化需求。

Jobdone結合國外頂尖CPM的產品理念與功能，打造獲得多項專利的平台架構，提供營造工程最多功能，最低入門成本，最具彈性整合的PMIS系統。從大型集團到小型開口合約，您都可以在平台上找到適合您的應用。
