# 專案分項工程

**「專案分項工程」**&#x529F;能允許使用者填寫並編輯專案中的各項子工程，便於詳細規劃和管理專案的各個施工階段與任務。透過此功能，使用者可以根據專案需求添加、修改或刪除分項工程，確保每個階段的工作都能夠精確跟蹤與管理。


!!! info
    若在公司通用設定中已設定🔗[**分項工程**](../../../company_configuration/subdivision)，會自動帶入到專案分項工程中。


![專案分項工程](../../../images/00c136dedb60ea1d.png)

***

### 01｜編輯 <a href="#edit" id="edit"></a>

如圖一，進入**專案分項工程**頁面後，點選右上角的 ![](https://2002560121-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEqUCL3D5WQfpxJw8NL3P%2Fuploads%2FXtKszvIUbA7fggMlMknl%2Fimage.png?alt=media\&token=5b142fcc-f710-42d9-88ac-891185399f82) 按鈕，可根據各專案需求，新增/刪除專案中的分項工程。

![](../../../images/5073d73e380f43d3.png)

如圖二，點選 ![](https://2002560121-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEqUCL3D5WQfpxJw8NL3P%2Fuploads%2FzEPhgz4ga8BubFE1lS3K%2Fimage.png?alt=media\&token=1d78129f-812e-4be2-ad6c-639c92f5bfad)，即可增加新的分項工程；亦可於欲刪除的分項工程旁，點選 ![](https://2002560121-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEqUCL3D5WQfpxJw8NL3P%2Fuploads%2FE2eVfbUxuhnpUsJSo5hl%2Fimage.png?alt=media\&token=b61834bf-5163-4e8e-8116-2a6313bb53d6) 圖示以刪除分項工程。


!!! warning
    刪除分項工程時，與該分項工程相關的所有資料（如進度、資源分配、報告等）將一併被刪除。請特別留意，操作前務必確認該分項工程不再需要，以免造成無法復原的資料損失。


![](../../../images/2d737cd1342b13e0.png)

如圖三，完成修改後，按下 ![](https://2002560121-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEqUCL3D5WQfpxJw8NL3P%2Fuploads%2FO3mzxYNp5UzBSOYmsXaM%2Fimage.png?alt=media\&token=4ea5bff6-0d5a-45ae-b928-274c0e78ad0d) 即可保存所做的變更並完成修改；若需放棄變更，按下 ![](https://2002560121-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEqUCL3D5WQfpxJw8NL3P%2Fuploads%2F0hDCk1tU9Ytbi4K3G2Fq%2Fimage.png?alt=media\&token=cd1f5298-2f22-45ac-b30e-868df7a40471) 即可恢復原有資料，無需儲存任何更動。

![圖三 - 儲存/取消](../../../images/10afe76c793a3374.png)
