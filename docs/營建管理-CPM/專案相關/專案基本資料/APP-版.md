# APP 版


!!! info
    在 **App** 上，使用者僅能查看專案資訊，無法進行編輯操作。


登入後，點&#x9078;**「專案清單」**，並選擇要查看的專案。進入專案後點選「專案資訊」。

<div>![點選「專案清單」](../../../images/ca1330f63f19b719.png) ![進入欲查看的專案](../../../images/e51ab4e51151d6f3.png) ![點選右上角之「專案資訊」](../../../images/10b494e5c0ca3868.png)</div>

點選專案基本資料中&#x4E4B;**「專案資訊」**，並可查看該專案基本資料。

<div>![點選專案基本資料之「專案資訊」](../../../images/4ab64c4d7cee79ec.png) ![查看專案資訊](../../../images/39ae05faed3d378e.png)</div>
