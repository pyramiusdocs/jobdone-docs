# 審核檢查紀錄

進入專案內的檢查表後，選&#x64C7;**「審核中」**&#x5206;頁即可看見待審核的檢查表。

點選下方&#x7684;**「審核通過」**&#x6216;**「退回」**&#x5373;可完成審核。


!!! warning
    系統預設自動勾&#x9078;**「僅顯示我的檢查工作」**，若有需要請關閉，避免找不到需查看的檢查工作。
