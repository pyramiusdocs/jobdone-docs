# 責任廠商列表

在驗收管理系統中，責任廠商列表提供以下兩種顯示模式，分別滿足不同的管理需求：


!!! info
    預設畫面將優先顯示『廠商列表』；如欲查看空間或戶別的缺失，請利用右上角的按鈕切換至『驗收標的』模式。
    
    * 當按鈕  ![](https://2002560121-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEqUCL3D5WQfpxJw8NL3P%2Fuploads%2FeA50NOPhOqRLZHNzLZ88%2Fimage.png?alt=media\&token=b985e5ca-8cbd-4302-80cb-2231c1bfc63e) 圖示時，表示目前處於廠商列表模式。
    * 當按鈕為 ![](https://2002560121-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEqUCL3D5WQfpxJw8NL3P%2Fuploads%2FGtlYL998TkfJLA4BU35e%2Fimage.png?alt=media\&token=2f0564e6-c2c5-4bae-b78a-e15b050a9b81) 圖示時，即可切換至驗收標的模式，按空間查看缺失項目。


**模式一：以『廠商』為核心**

此模式按廠商分類列出所有承攬單位，並直觀顯示各廠商的缺失總數。點選特定廠商後，即可查看該廠商負責的所有缺失項目細節。

![圖一 - 依廠商顯示](../../../../../images/60b03891a54b7338.png)

***

**模式二：以『戶別/空間』為核心（驗收標的模式）**

此模式按戶號排列，並於每一戶內詳細羅列各個空間（如客廳、臥室、衛浴）所涉及的各廠商缺失。

![圖二 - 驗收標的模式](../../../../../images/e6649bac2ea3086e.png)
