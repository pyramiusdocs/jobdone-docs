# 材料進出管制紀錄

Jobdone 系統全新推出「材料進出管制」模組，與****每日施工日誌****深度整合。本功能核心在於建立數位化的進貨單紀錄系統，讓現場工程師能針對各類建材進行詳盡的品檢與數量控管，確保工程用料符合設計規範與合約要求。

系統針對營建工程中的核心材料，設計了專屬的欄位與進貨證明上傳機制，以達到分門別類：

{% tabs %}
{% tab title="一般材料(General materials)" %}
適用於各類消耗品或零星資材，紀錄基礎進出廠數量與規格。
{% endtab %}

{% tab title="混凝土(Concrete)" %}
除了基本數量，系統依據混凝土特性與檢測規範，額外強化品管數據填寫，包含：抗壓強度、坍度(Slump)、水灰比(W/C)、氯離子含量檢測，以及關鍵的出場、交貨時間等。
{% endtab %}

{% tab title="鋼筋(Rebar)" %}
系統針對鋼筋的特性，設計了嚴謹的數位進場表單。除基本交貨資訊外，更強化了現場品檢與存放管裡的紀錄欄位，確保鋼筋從進場到綁紮前皆符合設計規範。

* 號數與規格：詳實記錄進場鋼筋之直徑編號（如：#3、#4、#10 等）及強度等級（如：SD280、SD420）。
* 材料標示：確認每批鋼筋是否掛有合格標籤（含製造廠名、熔煉號、規格等）。
* 堆置與保護狀況：紀錄現場存放環境，例如：是否墊高隔離地面 以及 是否覆蓋防塵布 或 防雨帆布（防止生鏽氧化）。
{% endtab %}

{% tab title="鋼構(Structural Steel)" %}
針對加工與組裝品質進行紀錄，涵蓋：表面狀況、切割面平整度、焊接點外觀、螺栓孔尺寸等。同時支援上傳「無輻射證明」、「出場檢驗證明 (Mill Report)」及「非破壞性檢測 (NDT) 報告」。
{% endtab %}
{% endtabs %}

### 02｜App 端

1. **App 端與網頁端的架構差異 (Mobile vs. Web)**

在操作邏輯上，您可以清楚感受到兩者的定位區隔：

{% tabs %}
{% tab title="網頁端" %}
將 材料進出管制紀錄 收納於「施工日誌」的更多事項中。這是為了讓內業人員在審核日報時，能在一條龍的介面中完成所有數據的檢視與核銷。
{% endtab %}

{% tab title="App 端" %}
將其設計為獨立的功能按鈕。

由於現場工程師在材料進場（如混凝土車抵達、鋼筋卸貨）時，往往需要第一時間處理。將功能獨立做一個入口，能讓您在手機上「一鍵進入」，無需先開啟日誌分頁，大幅縮短操作路徑。
{% endtab %}
{% endtabs %}

2. **App 端的新增與操作**

雖然入口位置不同，但 App 端的權限與功能依然完整：

* 快速新增進貨單：App 端優化了輸入介面，讓您在走動中也能輕鬆填寫。
* 即時拍照存證：App 端最大的優點是能直接啟動相機。在編列材料清單（如鋼筋、鋼構）時，拍完照片時立即上傳至該筆單據，比網頁端事後上傳照片更具效率。
* 數據同步機制： 在 App 端新增的任何進貨單與材料細節，會即時同步至網頁端的施工日誌中。換句話說，您在門口簽收一車混凝土，辦公室的內業人員打開網頁日誌就能立刻看到數據。
