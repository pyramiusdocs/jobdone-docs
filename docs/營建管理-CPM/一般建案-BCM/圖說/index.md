# 圖說

在工地現場，圖說的版本管理直接決定了施作的正確性。如果工班拿著舊版圖紙去放樣或施工，後續的拆改（Rework）成本將無法估算。圖說中心並非單純的檔案存放空間，而是專門針對圖說生命週期設計的管理機制。從最初的合約圖、現場施工套圖到後續的設計變更，每一張圖紙的每個版次（Revision）都有完整的數位軌跡與審核歷程。

這套系統建立了一套動態的版本控管標準。透過權限設定與標籤篩選，確保現場人員在手機上打開時，看到的絕對是最新且經過核可的施工依據，解決現場與辦公室資訊不對稱的問題，對於結案時的竣工圖面整理也有實質的管理幫助。

***

#### 圖說全週期管理

圖說列表詳實記錄了每一張圖說從無到有、從施工到竣工的所有狀態。透過清單式管理，工程人員可以精確掌握目前工地現場拿的是哪一個版本，並進行各階段的審核與發布動作。
