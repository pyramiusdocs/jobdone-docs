# App精簡日誌 / 基本資料

如下圖，進入<kbd>**基本資料**</kbd>頁籤，點選下&#x65B9;**「🖊️基本資料」**&#x5373;可開始填寫基本資料。

包括**開工狀態**、**表報編號**、**上下午天氣**及**實際進度**等。


!!! warning
    請注意，請先填寫基本資料再進行日誌內容(施工概況、材料概況......)填寫。
