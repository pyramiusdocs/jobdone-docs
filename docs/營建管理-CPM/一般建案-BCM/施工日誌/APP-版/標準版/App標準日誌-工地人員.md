# App標準日誌 / 工地人員

如下圖，於<kbd>**工地人員**</kbd>頁籤點選下方&#x4E4B;**「+工地人員」**， 即可選擇當日到場之工種(工別)，並填寫本日人數。

<div>![圖一 - 進入工地人員頁面](../../../../../images/80daa0122f7b249f.png) ![圖二 - 選取工種(工別)](../../../../../images/579f618526c38244.png) ![圖三 - 填寫該工種本日到場人數](../../../../../images/ec073582d26901ba.png)</div>

將資料填寫完畢後，點選圖四下方&#x4E4B;**「確定新增」**&#x5373;可見(圖五)畫面。

如需更動工地人員資料，點選該工種後即可見(圖六)畫面，修改**本日人數**及**刪除工地人員**。

修改完畢並確認資料無誤後，按&#x4E0B;**「確定新增」**&#x5373;完成資料更動。

<div>![圖四 - 確定新增/取消](../../../../../images/b6d557796d3747d6.png) ![圖五 - 新增成功畫面](../../../../../images/dff81c1f78dec7c2.png) ![圖六 - 更改工種本日人數/刪除](../../../../../images/1e3e0265daf523de.png)</div>
