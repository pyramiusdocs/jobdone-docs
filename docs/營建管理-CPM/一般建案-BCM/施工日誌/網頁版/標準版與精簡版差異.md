# 標準版與精簡版差異

### **🔹 標準版施工日誌**

在標準版中，您將根據先前已建立好之專案資料進行詳盡填寫。具體來說，以下是標準版的主要特點：

#### **自動帶入專案設定資料**

當您填寫每日施工日誌時，**施工概況**、**工地材料管理概況**、**工地人員管理**以及**機具管理**等項目，都會請您自行選擇當日施作/使用的項目，選擇列表將自動對應到您在專案資料中事先設定好的內容。

以施工概況為例：系統會依據&#x65BC;**「施工項目」**&#x586B;寫之資料，自動顯示該項目的**單位**、**契約數量**、是否需要**專業技術士**、是否需填寫**自主檢查表**、是否有**檢驗停留點**等詳細資訊。

> **施工概況**  **➙** 專案資料設定 - [shi-gong-xiang-mu](../../../project_level/zhuan-an-zi-liao-she-ding/shi-gong-xiang-mu "mention")
>
> **工地材料管理概況**  **➙** 專案資料設定 - [material\_management](../../../project_level/project_data/material_management "mention")
>
> **工地人員管理**  **➙** 專案資料設定 - [trade-category](../../../project_level/project_data/trade-category "mention")
>
> **機具管理**  **➙** 公司通用資料設定 - [equipment](../../../../company_configuration/equipment "mention")

***

#### 進度與完成數量計算（於每日日誌內）

填寫<kbd>**施工概況**</kbd>時，會需要您針對各施工項目選擇其是否&#x70BA;**「一式」**&#x4E4B;項目。

將因應是否為一式而有些許差異：

{% tabs %}
{% tab title="一式項目" %}
如果施工項目為一式項目，您不必填寫當日完成數量，但需要手動填寫累積完成進度。
{% endtab %}

{% tab title="非一式項目" %}
若為非一式項目，您只需填寫當日完成的數量，系統會自動計算並更新累積完成進度。
{% endtab %}
{% endtabs %}

填寫材料使用、機具使用、出工概況時，您一樣只需填寫本日使用數量/本日人數，系統即會自動幫您計算累積使用數量/累積出工人數。

***

#### **進度與使用情況詳細資料（於各功能內）**

系統會自動計算並將各資料記錄於：「**工項施作進度」**、**「材料使用概況」**、**「機具使用概況」**&#x53CA;**「出工概況」**。

{% tabs %}
{% tab title="工項施作進度" %}
共&#x6709;**「工項總表」**、**「分項工程」**&#x8207;**「專案合約」**&#x4E09;種檢視模式，系統會根據日誌紀錄 (完成數量與進度)，自動計算各分項工程其下**施工項目之進度**、各**分項工程之進度**、**專案整體施工進度**。

除**單位**、**契約數量**、**單價**、**複價**等基本資料，系統亦會根據各工項複價自動計算其**權重**。可詳細查看各工項之**完成數量**、**完成進度**及**已施工天數**等。並可於施工日期中，點選任一日期查看當日日誌紀錄。
{% endtab %}

{% tab title="材料使用概況" %}
包括所有材料之**單位**、**契約數量**、**單價**、**複價**與**使用總量**等基本資料。

亦可詳細查看各項材料之**完成數量**、**完成進度**及**使用紀錄**等。並可於使用日期中，點選任一日期查看當日日誌紀錄。
{% endtab %}

{% tab title="機具使用概況" %}
包括所有機具&#x4E4B;**「累計數」**&#x8207;**「使用日期」**。並可於使用日期中，點選任一日期查看當日日誌紀錄。
{% endtab %}

{% tab title="出工概況" %}
包括所有工種&#x4E4B;**「預計工數」**、**「累積工數」**&#x8207;**「出工日期」**。並可於出工日期中，點選任一日期查看當日日誌紀錄。
{% endtab %}
{% endtabs %}

***

#### 施工進度圖表顯示

系統會根據日誌紀錄，對每一分項工程進度進行個別計算，再自動計算**專案的整體施工進度**，將進度數據以**圖表**方式展示，並顯示**實際進度**與**預期進度**之間的差異，便於項目管理層掌握施工狀況並調整計劃。

***

### 🔸 **精簡版施工日誌**&#x20;

精簡版日誌相比標準版，功能有所簡化，主要特點如下：

#### **無需套用專案設定資料**

精簡版日誌不會自動套用事先在專案資料中設定的內容，因此不會顯示**單位**、**契約數量**等詳細資料。所有項目（包括施工項目、工種、材料及機具等）均需由使用者自行手動填寫。

***

#### **無一式項目選擇**

精簡版中，施工項目無法選擇是否為一式項目。使用者需要直接填寫**當日完成數量**以及**累積完成數量**，並且系統不會自動計算進度。

***

#### **無進度與使用情況自動計算**

在精簡版中，系統不會自動幫忙計算或記錄**工項施作進度**、**材料使用概況**、**機具使用概況**等數據，也不會記錄**出工概況**。所有這些資料需要由使用者自行處理。（即沒有以上功能）

***

#### **無進度圖表展示**

精簡版不提供進度對比圖表，因此您無法看到**實際進度**與**預期進度**之間的差異，也無法通過圖表直觀了解工程進度。

***
