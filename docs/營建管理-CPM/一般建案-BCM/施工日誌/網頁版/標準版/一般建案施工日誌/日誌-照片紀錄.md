# 日誌 / 照片紀錄


!!! info
    在填寫日誌的照片紀錄之前，必須先完成基本資料的填寫。


![標準版 - 照片紀錄](../../../../../../images/cece72eecf566312.png)

***

## 選取照片


!!! warning
    此處照片由當日影音紀錄上傳之圖片選取。即，**當日必須有**影音日誌紀錄才能於此選擇圖片。
    
    影音日誌相關操作，請參閱[ ](../../../../media_diary) **➙** 🔗 [影音日誌](../../../../media_diary)


於照片紀錄欄位，如(左圖)紅框圈選處，點&#x9078;**「**&#xD83D;?️ **照片紀錄」**，即可進入照片選取頁面(右圖)。

可點選照片查看並勾選，勾選完畢並確認無誤後，點&#x9078;**「儲存變更」**&#x5373;可保留資料。

<div>![](../../../../../../images/025062279778430b.png) ![右圖 - 進入選取頁面](../../../../../../images/8895b09a0d270b6e.png)</div>
