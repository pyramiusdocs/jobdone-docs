# 影音日誌

將分&#x70BA;**「網頁版」**&#x8207;**「APP 版」**&#x5169;種說明。兩者功能上不太相同，操作介面亦略有不同。


!!! warning
    請注意，僅能新增於專案日期內 (開工-完工日期/展延日期) 之日誌紀錄。


在營建管理中，影音日誌不只是「拍照片存檔」，它是現場狀況的第一手證言。無論是自主檢查、施工前中後的紀錄，還是單純的工項交接，影像紀錄能直接彌補文字無法表達的細節。本功能整合了 App 的機動性與 Web 端的大量管理需求，讓現場的每一張照片、每一段影片，都能自動對應到正確的工程位置與責任包商，建立一套完整的工程履歷。

本系統的設計核心在於「結構化紀錄」，將混亂的工地影像轉化為有序的管理單元，避免因人員更迭或時隔久遠，導致找不到照片或對不上工序的困擾。

工地每天產生的影像成百上千，若沒有分類，日後查核就是大海撈針。系統採用與實務對齊的分類架構，確保每一則影音都能精準掛載標籤：

{% tabs %}
{% tab title="App 端：即時擷取與分類" %}
現場工程師或監造人員可利用 App 即時拍攝施工現況，並利用下拉式清單快速選取標籤。支援短影音功能，能記錄單靠照片難以表達的動態過程（如：漏水測試、管線通孔測驗、設備運轉測試）。紀錄完成後即刻同步雲端，避免因手機遺失或群組照片過期導致的重要數據流失。
{% endtab %}

{% tab title="Web 端：高效檢索與管理" %}
管理人員在辦公室透過網頁端，可利用複合篩選條件（如：日期 + 協力廠商 + 施工項目）快速定位特定紀錄。系統能根據掛載的標籤自動彙整影像流，讓管理者無需親臨現場，也能透過螢幕進行「虛擬巡檢」，掌握各案場、各區域的真實施工品質。
{% endtab %}
{% endtabs %}


!!! info
    #### 功能特點
    
    1. 強化界面透明度： 當土建、機電、裝修多方在同一區域交錯施工時，透過系統標籤過濾功能，各方能即時查閱前項工序的完工影像，降低界面溝通成本，減少重工發生。
    2. 縮短查核時間： 針對隱蔽工程（如澆置前查驗）或材料驗收，管理人員能快速調閱具備時間戳記與標籤的影音紀錄，大幅提升內部稽核與進度確認的效率。
    
    本系統的影音日誌功能，是將現場瑣碎影像轉化為專業管理數據的橋樑。它透過嚴謹的五大標籤架構，確保每一筆施工紀錄都能「找得到、對得上、說得清」，為工程團隊提供一個透明、專業且具備回溯價值的數位管理平台。
