# LINE 群組助理

#### 01｜簡介

「LINE 群組助理」是專為營建與工程管理量身打造的智能紀錄工具。針對工地現場溝通頻繁、資訊碎片化的特性，本工具能自動化保存 LINE 群組內的施工照片、會勘紀錄及各方指令，將非正式的聊天內容轉化為具備追蹤價值的專案資產。

無論是分包商的進度回報、業主的變更指示，或是職安檢查的缺失紀錄，群組助理都能確保這些「數位證據」永不失效，解決營建業最常見的資訊斷層問題。


!!! info
    #### 為什麼工程專案需要它？
    
    釐清施工責任，假使下包商聲稱「當初在 LINE 群組說過要改色」，但對話早被洗掉，圖片也打不開。有了群組助理，管理者只要搜尋關鍵字，立刻就能調出當時的對話截圖與時間戳記，讓數據說話，終結互踢皮球。


**💡 營建實務應用場景建議**

#### 02｜功能特點


!!! warning
    #### 請注意
    
    「LINE 群組助理」的管理與檢索介面僅限於「網頁版」環境下使用。此設計旨在提供管理者更寬廣的視覺範圍，方便在電腦端進行大量施工照片的快速檢閱、複雜對話的精準篩選，以及工程檔案的批量下載。
