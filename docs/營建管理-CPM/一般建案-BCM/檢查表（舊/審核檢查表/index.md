# 審核檢查表

將分&#x70BA;**「網頁版」**&#x8207;**「APP 版」**&#x5169;種說明。兩者功能大致相同，僅操作介面略有不同。

***

檢查表的審核流程分&#x70BA;**「初驗審核」**&#x8207;**「複驗審核 」** 兩種情形，兩者的操作方式與概念基本相同。

&#x5728;**「審核中」**&#x968E;段，您將查&#x770B;**「待初驗審核」**&#x7684;檢查項目；&#x5728;**「複驗審核」**&#x968E;段，則會查&#x770B;**「待複驗審核」**&#x7684;檢查項目。

審核者可根據檢查結果作出以下兩種評斷：

{% tabs %}
{% tab title="退回修改" %}
要求檢查人員針對問題進行修正。
{% endtab %}

{% tab title="通過並簽名" %}
確認檢查項目符合標準並完成審核程序。
{% endtab %}
{% endtabs %}


!!! tip
    此流程確保檢查項目在每個階段均獲得有效的審核與確認。
