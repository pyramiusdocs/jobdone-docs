# 施工日誌

## 相關設定

填寫施工日誌時，可能會使用到[**施工項目**](../../project_level/project_data/construction_item)、**材料管理**、[**專案工種(工別)類型**](../../project_level/project_data/trade-category)、[**施工項目總表**](../../project_level/project_data/general_list_of_construction_items) 等設定。\
會需要您先於 **➙**[ **專案資料設定**](../../project_level/project_data) 頁面中進行設定。

![](../../../../images/7ce7193454890ce5.png)

## 啟用功能

建立專案時，施工日誌功能預設會是啟用狀態，專案經理可開啟或關閉功能。

![](../../../../images/f775473ae33aba41.png)

## 日誌資訊一覽

![](../../../../images/19fc7ca5d550984c.png)

### 日誌模式

顯示當前選用的日誌模式 。分為 「 **公共工程施工日誌 」** 及 「 **建築物施工日誌 」**，可以在**基本設定**中切換兩種模式。

### 專案工期資訊

**「 總工期 」** 可在[**專案設定**](../../project_level/basic-information)中設定。「 **累計工期 」**、「 **不計工期 」**、「 **剩餘工期 」** 由系統根據施工日誌填寫的內容進行自動計算。

### 實際進度

依據日誌中填寫的工項施工數量，為您計算您施工的總進度。


!!! info
    施工總進度的計算方式為：**完成的工項金額總額 ÷ 總工項金額 = 實際進度**


### 預計進度

預計進度由系統自動計算生成，百分比取至小數點後兩位。如希望自定進度計畫，可以前往[**預計進度設定**](ri-zhi-xi-tong-she-ding/yu-ji-jin-du-she-ding)進行設定。

## 日誌月報

日誌月報可迅速瀏覽每天日誌的填寫概況，也可以點選日期查看指定日期的日誌報告。
