# 預計進度設定

## 建立進度計畫

> 1.  點選畫面右側偏上方的 **建立按鈕**
>
>     ![](../../../../images/c8f8856724ad7439.png)
>
>
> 2.  輸入 **日期** 與該日期 **進度計畫**。
>
>     ![](../../../../images/51120a7dd871e713.png)
> 3.  按下 **建立進度設定** 按鈕。
>
>     ![](../../../../images/9c1dfe733649aa00.png)
>
>
> 4.  建立進度計畫成功！
>
>     ![](../../../../images/19c98de6cfc43fa9.png)
>
>
