# 基本設定

## 日誌模式設定

Jobdone 提供兩種公共工程日誌模式。分別為 「 **公共工程施工日誌** 」 與 「 **建築物施工日誌** 」 。\
差別在於兩者個 **列印格式不同**。

### 如何切換日誌模式?


!!! tip
    切換日誌模式並不會造成日誌資料遺失。


> 1.  點選右側的  日誌模式 按鈕。<br>
>
>     ![](../../../../images/5d2b2b3c1cea1ba8.png)
>
>
> 2. 勾選 您需要模式。\
>    ![](https://content.gitbook.com/content/EqUCL3D5WQfpxJw8NL3P/blobs/Tf0YuvDwBEl94D9t6DI9/image.png)<br>
> 3. 點選 儲存變更。\
>    ![](https://content.gitbook.com/content/EqUCL3D5WQfpxJw8NL3P/blobs/8ZWjral9jiSU4uc2ZH8p/image.png)<br>
> 4. 日誌模式變更成功！\
>    ![](https://content.gitbook.com/content/EqUCL3D5WQfpxJw8NL3P/blobs/5qL9eyRLUxqlUWzssitN/image.png)
