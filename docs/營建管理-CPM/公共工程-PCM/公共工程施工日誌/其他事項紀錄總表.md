# 其他事項紀錄總表

## 📓這些紀錄那裏來的?

紀錄可於 **本頁面進行新增**，或從 [日誌當中的 → 日誌 / 其他更多紀錄 ](shi-gong-ri-zhi/ri-zhi-qi-ta-geng-duo-ji-lu)新增。



## 📓列表篩選

Jobdone提供 **日期區間** 篩選功能。

![日期區間 篩選功能](../../../images/5acf7b864678f24f.png)



## 📓釘選紀錄(將記錄置頂)

如有重要紀錄需要保留於列表的最上方，可 點選列表左側的**星星**進行釘選。再次點選擇取消釘選。

<div align="left">![釘選中狀態](../../../images/60044535854cbefe.png) ![取消釘選狀態](../../../images/53b598b8601c015e.png)</div>



## 📓01｜本頁面如何新增紀錄?

* 介面右上方有個 **新增紀錄** 的按鈕  ( 左圖🔴)
* 點選即可開啟編輯頁面 ( 右圖 ) 。
* 填寫完畢後 點選右下角的 **新增紀錄** ( 右圖🔴) 即可新增紀錄。

<div>![](../../../images/f94057e496b2bbbc.png) ![](../../../images/fc329917ad9fe906.png)</div>



## 📓03｜編輯、刪除紀錄

找到您要操作的提示項目，於該項目的最右側，有個 **三個點圖案的按鈕**。點選後會出現 **編輯** 與 **刪除** 的按鈕。

* 刪除：請點選刪除按鈕。
* 編輯：請點選編輯按鈕。並於修改介面中修改完成後按下儲存按鈕。
