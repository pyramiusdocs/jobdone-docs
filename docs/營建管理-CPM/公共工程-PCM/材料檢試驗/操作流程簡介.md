# 操作流程簡介

{% stepper %}
{% step %}
### 確保「材料管理」資料已妥善填寫完畢

如下左圖，進&#x5165;**「材料管理」**&#x9801;面查看所有材料，並針對需求點選右上方&#x4E4B;**「新增」**&#x8207;**「編輯本頁」**。       &#x20;

詳細操作流程，請參閱 ➙ [material\_management](../../project_level/project_data/material_management "mention")


!!! tip
    由於編列材料設備管制總表時，會需要選擇&#x65BC;**「材料管理」**&#x6240;建立的各項材料，因此務必確認資料無誤。


<div>![進入「材料管理」頁面](../../../images/75b11979e3c9b734.png) ![新增/編輯本頁](../../../images/ab3eff7de0c7e803.png)</div>

<div>![新增材料](../../../images/5949c3a670ab5ef2.png) ![編輯材料](../../../images/0b42a2f4cf6c8b51.png)</div>
{% endstep %}

{% step %}
### 上傳品質管理標準

進入材料設備管制總表後，點選右上角&#x4E4B;**「**![](../../../images/8035d4ba3402184e.png)」，即可上傳/編輯您的相關檔案。


!!! tip
    針對檢驗材料，您可能需上傳品質管理標準，以利檢驗時即時查看，並確保檢驗流程符合規範標準。


<div>![進入「材料設備管制總表」](../../../images/5f19078f3b783ff7.png) ![點選「品質管理標準」](../../../images/73779687b4de3fac.png)</div>

點&#x9078;**「上傳檔案」**&#x5373;可開啟新增視窗，選擇欲上傳之品質管理標準。

<div>![點選「上傳檔案」新增檔案](../../../images/723e8257ae5b2825.png) ![](../../../images/eed23c44f0eb751b.png)</div>

上傳成功並確認資料無誤後，點&#x9078;**「儲存」**&#x5373;完成上傳。

<div>![儲存/取消](../../../images/f1df485cc4c9e0dc.png) ![上傳成功](../../../images/03c66c909e653fc2.png)</div>
{% endstep %}

{% step %}
### 選取管制總表之材料

點&#x9078;**「**![](../../../images/6e2e7bba44fb68ce.png)**」**，即可開啟材料篩選器。開啟材料篩選器並完成篩選後，即可勾選材料至管制列表中。


!!! warning
    請注意，僅能勾選尚未於列表內之項目。


<div>![點選「選擇材料」](../../../images/f25b488940da6e38.png) ![勾選材料至管制列表](../../../images/0a00d9580e085919.png)</div>
{% endstep %}

{% step %}
### 申請檢驗

於管制列表中勾選材料，即可點選右上角&#x4E4B;**「+申請檢驗」**，開始為選取材料進行檢驗。

<div>![勾選列表中之材料](../../../images/190cc63d70416a4b.png) ![點選「+申請檢驗」](../../../images/8bf5f499682e6c21.png)</div>

選妥檢驗項目後，即可開始填寫該檢試驗之**預定進場日期**、**預定檢驗日期**及**檢驗地點**等等，確認送出後，您即可於 ➙ [rr](rr "mention") 查看該張檢試驗申請單。

![填寫檢試驗申請單](../../../images/3819323bf78ae18a.png)
{% endstep %}

{% step %}
### 列印檢驗申請單

進入「<kbd>**檢試驗申請紀錄**</kbd>」，於欲列印之檢試驗申請單右側操作欄位，點&#x9078;**「列印申請單」**。

<div>![點選「列印申請單」](../../../images/585932948b85b130.png) ![列印畫面預覽](../../../images/99b20c1e6c775a12.png)</div>
{% endstep %}

{% step %}

詳細操作流程，請參閱 ➙ [rr](rr "mention")
{% endstep %}
