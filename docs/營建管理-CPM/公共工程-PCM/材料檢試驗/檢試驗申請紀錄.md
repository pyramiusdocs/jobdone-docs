# 檢試驗申請紀錄

## 01｜編輯檢試驗紀錄

於檢試驗申請紀錄列表中，選取欲編輯之申請單。點&#x9078;**「**![](../../../images/12d9970a9445914d.png)」即可查看細部內容，並針對各項檢驗材料填寫檢試驗紀錄。

選定申請單後，於內部檢驗材料設備之右側。點&#x9078;**「編輯紀錄」**，即可填寫檢試驗紀錄。

<div>![開啟申請單](../../../images/88607d0be39c8253.png) ![於各檢驗材料旁，點選「編輯」](../../../images/9d7522cb2eaa0f61.png)</div>

系統將紀錄填寫分為兩個步驟，<kbd>**取樣**</kbd>**&#x20;&&#x20;**<kbd>**回填檢驗結果**</kbd>&#x20;


!!! tip
    您可於材料取樣檢驗後，先行填&#x5BEB;**「取樣」**&#x8CC7;料 (無需同時填寫)。待檢驗結果出爐，再回填材料檢驗是否合格。


{% tabs %}
{% tab title="取樣 (含送驗)" %}
取樣資料包括：實際進場數量與日期、檢視方式、取樣項目與日期、送樣方式與日期等等。
{% endtab %}

{% tab title="回填檢驗結果" %}
回填檢驗結果檢驗結果資料包括：試驗日期、會驗人員、試驗機構名稱、營造初判結果、監造複判結果等等。
{% endtab %}
{% endtabs %}

<div>![取樣 (含送驗)](../../../images/4acf50cd435278d3.png) ![回填檢驗結果](../../../images/c8fa1b2541c0b135.png)</div>

## 02｜檢試驗申請單

### 02 - 1｜列印

進入「<kbd>**檢試驗申請紀錄**</kbd>」，於欲列印之檢試驗申請單右側操作欄位，點&#x9078;**「列印申請單」**。

<div>![點選「列印申請單」](../../../images/42e945f0aabc72b3.png) ![列印畫面預覽](../../../images/99b20c1e6c775a12.png)</div>


!!! tip
    將檢試驗申請單印出後，就可以將紙本文件提交給監造單位審核。
    
    列印格式提供兩個欄位：**營造廠商** & **監造廠商**。供廠商簽認用。


![簽認欄位](../../../images/22c286c5d9a61ee4.png)

### 02 - 2｜刪除

進入「<kbd>**檢試驗申請紀錄**</kbd>」，於欲列印之檢試驗申請單右側操作欄位，點&#x9078;**「刪除申請單」**。


!!! warning
    請注意，包含該申請單內所有材料之檢驗紀錄都將一併刪除，且刪除後即無法復原，務必謹慎操作。


<div>![點選「刪除申請單」](../../../images/305b1664ad33f8b2.png) ![刪除申請](../../../images/87ef0ea1c1d8bc78.png)</div>

***

## 03｜檢試驗紀錄

### 03 - 1｜列印

開啟檢試驗申請單後，於欲列印之檢驗材料右側。點&#x9078;**「⋮」**&#x4E4B;**「列印檢試驗紀錄」**，您即可列印單內檢驗材料之檢試驗紀錄/結果。

<div>![點選「列印檢試驗紀錄」](../../../images/2406b1bc317a6d47.png) ![列印預覽](../../../images/5a7d47d59026955d.png)</div>


!!! tip
    列印格式提供兩個欄位：**營造廠商** & **監造廠商**。供廠商簽認用。


![簽認欄位](../../../images/b26442754034cbe7.png)

### 03 - 2｜刪除

開啟檢試驗申請單後，於欲刪除之檢驗材料右側。點&#x9078;**「⋮」**&#x4E4B;**「刪除檢試驗紀錄」**。


!!! warning
    請注意，選定材料之檢驗紀錄都將一併刪除，且刪除後即無法復原，務必謹慎操作。


<div>![點選「刪除檢試驗紀錄」](../../../images/b76eb3e0c3ec98c6.png) ![確認/取消刪除](../../../images/63cfc41bbec8ea34.png)</div>

***

## 04｜建立改善單

無論是營造初判或是監造複判之結果，針對**不合格**的材料檢驗紀錄，即可直接建立改善單。

<div>![點選「建立改善單」](../../../images/d6e89d390a7d3eb0.png) ![填寫改善單](../../../images/26992f0c363c2a4d.png)</div>

若一不合格材料已建立其改善單，可直接查看<kbd>**改善單進度**</kbd> / <kbd>**重新發起改善**</kbd>。

<div>![查看改善單進度](../../../images/4ddd7acb5c053b44.png) ![重新發起改善](../../../images/fa6ebedbace189a3.png)</div>

***

## 05｜申請單狀態說明

系統將申請單分為<kbd>**進行中**</kbd>與<kbd>**已完成**</kbd>兩種狀態。

{% tabs %}
{% tab title="進行中" %}
材料檢驗結果尚未填寫完成。所&#x6709;**「未填寫」**&#x53CA;**「僅填寫取樣」**&#x8CC7;料的材料皆會顯示此狀態。、
{% endtab %}

{% tab title="已完成" %}
已將所有材料&#x7684;**「送驗」**&#x8CC7;料&#x53CA;**「回填檢驗結果」**&#x8CC7;料填寫完畢，申請單才會顯示此狀態。
{% endtab %}
{% endtabs %}
