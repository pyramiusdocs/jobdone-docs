## 01｜檢驗流程

### 01 - 1｜選擇材料

於材料設備管制總表中點&#x64CA;**「**![](../../../images/6e2e7bba44fb68ce.png)**」**，即可開啟材料篩選器，選擇相應的協力廠商，繼而增添尚未加入管制列表的材料。

<div>![點選「選擇材料」](../../../images/cea9ef24b7aeb6bb.png) ![材料篩選器](../../../images/92dca21f213e136f.png)</div>

勾選材料加入您的管制列表內。

![勾選材料至管制列表](../../../images/980f28724fc64e3b.png)

***

### 01 - 2｜申請檢驗

勾選欲申請檢驗的材料，再於右上方點&#x64CA;**「+申請檢驗」**，即可開始填寫檢試驗申請單。

<div>![勾選列表中之材料](../../../images/ba325016412ebaae.png) ![點選「+申請檢驗」](../../../images/8bf5f499682e6c21.png)</div>

系統將顯示申請檢驗的材料目前累積的檢驗次數。

您需填寫以下資料申請檢驗單，包括：申請日期、預定進場日期、預定取樣時間、取樣地點及檢附資料等。

![填寫檢試驗申請單](../../../images/2ac09313cb471822.png)

***

## 02｜材料篩選 & 檢視

### 02 - 1｜篩選材料

當您的材料數量過多時，可透過材料篩選器幫您快速找到欲查看/使用的材料。

點選下圖紅框處&#x4E4B;**「篩選」**，即可開啟材料器，透過<kbd>**是否取樣試驗**</kbd>、<kbd>**預定進場日**</kbd>、<kbd>**實際進場日**</kbd> 等查找。

<div>![點選「篩選」](../../../images/92219ccae1e53b97.png) ![材料篩選器](../../../images/64ee1074cd114878.png)</div>

### 02 - 2｜檢視方式

點選下圖紅框處&#x4E4B;**「檢視方式」**，即可選擇檢視模式。

可選擇全部或是<kbd>**送審管制表**</kbd>、<kbd>**檢試驗管制表**</kbd>兩種模式。

{% tabs %}
{% tab title="送審管制表" %}
僅顯示送審相關資訊。如：驗廠日期、預定/實際送審日、預定送審資料及使用位置等等。
{% endtab %}

{% tab title="檢試驗管制表" %}
僅顯示檢試驗相關資訊。如：預定/實際進場日、進場數量、抽樣/送樣日期與數量等等。
{% endtab %}
{% endtabs %}
