# 修改密碼

***


!!! warning
    請注意：密碼僅能在網頁版中進行修改。




#### 一、進入個人資訊頁面，點選 「 修改密碼 」 。

![](images/80bf55b8ea8783f2.png)

![](images/f60c96f39c2fba86.png)

#### 二、輸入想要變更的密碼，輸入完畢後點選 「 變更密碼 」。

![](images/425fa71e1f5a85f2.png)

#### 三、跳出 「 密碼更新成功 」 即可。

![](images/e6b5deb93de52eb3.png)
