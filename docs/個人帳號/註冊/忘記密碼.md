# 忘記密碼

除了找回帳號，若您遺失了密碼，Jobdone 也提供安全的密碼重設機制。系統會透過您帳號所綁定的 Email 信箱來協助您找回。

### 02｜App 端

除了電腦網頁版，Jobdone App 同樣提供便捷的重設密碼功能，操作流程如下：

1. 開啟 App：進入 Jobdone App 的登入畫面。
2. 觸發功能：於『密碼』輸入欄位處，點選 ![](https://2002560121-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEqUCL3D5WQfpxJw8NL3P%2Fuploads%2FSkjpVsEnSSjK9f22Ahis%2Fimage.png?alt=media\&token=4eddc11a-04bb-484a-83bb-bed22a4c10a4)。
3. 輸入帳號：輸入您註冊 Jobdone 時的帳號 ID，並點選 ![](https://2002560121-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEqUCL3D5WQfpxJw8NL3P%2Fuploads%2FGFfhMZpg9u0o08BIAv8m%2Fimage.png?alt=media\&token=2184722f-171d-427d-b8f7-de3591aa281a)。
4. 查收信件：您即可於電子郵件信箱中找到系統寄送的信件，並重新設定您的密碼。
