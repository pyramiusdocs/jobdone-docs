# 我的派工單

派工單功能用於指派個人負責執行的工作。

您可以在 「 我的派工單 」 中找到別人指派給您的工作項目及內容。



## 查看派工單

### 網頁版

1. 登入後，在左側欄位中點選 「 我的派工單 」。

![](images/7bb621f3afe586ef.png)

2. 點選上方 「 指派給我 」 及 「 由我發單 」 ，即可查看不同分類的派工單列表。

![](images/ece09d13f9a10fac.png)

### APP

1. 登入手機APP後，點選 「 我的待辦事項 」。
2. 點選上方 「 派工單 」 分類，即可查看派工單列表。

## 派工單回報及轉發

派工單回報後就結束，若回報的結果不被接受，則發起人須重新發起新的派工單。

接單者若判斷該工單應由其它人執行，可使用轉發功能。

### 網頁版

1. 進入派工單內容，點選 「 編輯 」，依派工單需求填寫完畢，點選 「 儲存 」
2. 儲存完畢後，即可點選右上角 「 回報工單並結案 」 送出。

* 若接單者判斷該工單應由其它人執行，則可點選左上角 「 轉發給其他人 」。

![](images/eddf4589a7a94c9f.png)

### APP

進入派工單內容，點選 「 編輯 」，依派工單需求填寫完畢後，即可點選下方 「 回報工單並結案 」 送出。若接單者判斷該工單應由其它人執行，則可點選下方 「 轉發給其他人 」。

![](images/a637cec7d93f2dd7.png)
