# 填寫改善單

***

## 網頁版

### 進入改善單

* 使用者登入帳號後，在左側選單點選 「 我的改善單 」 進入改善單列表。

![](images/6b9a2d6e892740f6.png)

### 改善單填寫

* 進入改善單列表後，點選改善單，填寫改善計畫／改善回報。

![](images/b10dedd50eb253e2.png)

![](images/618eabe96c8d4b2a.png)

### 填寫完畢

* 若改善單不會馬上送出，可點選 「 儲存 」 暫不發送。

![](images/69a865124c504aef.png)

* 若改善單不需內部審核，可直接點選 「 完成並送出改善回報 」 。

![](images/99ee84e599e58bb6.png)

* 若改善單須經內部審核

1. 點選 「 完成改善計畫，內部審核 」 ，即可選擇審核人，送出後開始進行內部審核。
2. 內部審核通過後，即可送出改善計畫／改善回報給發文方。

![](images/749a1fc26248f0a3.png)

<div align="left">![](images/553f845e412ba245.png)</div>

![](images/ab5b802bb7adbf46.png)

![](images/856d5c7f6cba8987.png)
