# 填寫改善單

***

### 網頁版

登入系統首頁後，於左側導覽列的<kbd>**我的待辦事項**</kbd>欄位中點選<kbd>**我的改善單**</kbd>，即可進入專屬管理頁面。在此您可以一目了然地查閱所有與您相關的單據，包含待辦缺失、待您審核等即時狀態。

![](../../images/b86b94a33b57a50f.png)

### 改善單填寫

* 進入改善單列表後，點選改善單，填寫改善計畫／改善回報。

![](../../images/dcf00a0a9ea5f66e.png)

![](../../images/57fa3cb6422e8412.png)

### 填寫完畢

* 若改善單不會馬上送出，可點選 「 儲存 」 暫不發送。

![](../../images/52a67e2e161da49f.png)

* 若改善單不需內部審核，可直接點選 「 完成並送出改善回報 」 。

![](../../images/5f66ca60f3aa2a59.png)

* 若改善單須經內部審核

1. 點選 「 完成改善計畫，內部審核 」 ，即可選擇審核人，送出後開始進行內部審核。
2. 內部審核通過後，即可送出改善計畫／改善回報給發文方。

![](../../images/d8d297130dadd049.png)

<div align="left">![](../../images/502e72140a7c9029.png)</div>

![](../../images/6da7b244233195b7.png)

![](../../images/e9d324153960ea55.png)

## App

### 進入改善單

* 登入App後，進入 「 我的待辦事項 」 ，在 「 待我處理 」 分頁中可查看未處理的改善單。

\
![](https://content.gitbook.com/content/EqUCL3D5WQfpxJw8NL3P/blobs/NKqvolLV750gA9TYLZan/%E6%94%B9%E5%96%84%E5%96%AE%E5%9B%9E%E5%A0%B1-1.jpg)  ![](https://content.gitbook.com/content/EqUCL3D5WQfpxJw8NL3P/blobs/eziRBjeko9y0rn7nsvLb/%E6%94%B9%E5%96%84%E5%96%AE%E5%9B%9E%E5%A0%B1-2.jpg)<br>

### 改善單填寫

* 點選 「 完整資訊 」 可查看該張改善單的相關資訊。

\
![](https://content.gitbook.com/content/EqUCL3D5WQfpxJw8NL3P/blobs/S5I2ATHK1mq3FZOm5o8j/%E6%94%B9%E5%96%84%E5%96%AE%E5%9B%9E%E5%A0%B1-3.jpg)  ![](https://content.gitbook.com/content/EqUCL3D5WQfpxJw8NL3P/blobs/S1wzeczGcYOn1sNQujEi/%E6%94%B9%E5%96%84%E5%96%AE%E5%9B%9E%E5%A0%B1-4.jpg)<br>

### 填寫完畢後

* #### 若改善單需送交內部審核

1. 點選 「 填寫完成 」 及可送交內部審核。

\
![](https://content.gitbook.com/content/EqUCL3D5WQfpxJw8NL3P/blobs/J5Hv9b44hSNwa483Fk3i/%E6%94%B9%E5%96%84%E5%96%AE%E5%9B%9E%E5%A0%B1-8.jpg)  ![](../../images/df9c3516ba1719f3.jpg)  \
![](https://content.gitbook.com/content/EqUCL3D5WQfpxJw8NL3P/blobs/wKJXYwqIUOj2lDRv2wwd/%E6%94%B9%E5%96%84%E5%96%AE%E5%9B%9E%E5%A0%B1-10.PNG) ![](https://content.gitbook.com/content/EqUCL3D5WQfpxJw8NL3P/blobs/pxgXeqgur2OAWPgJ1olR/%E6%94%B9%E5%96%84%E5%96%AE%E5%9B%9E%E5%A0%B1-11.jpg) <br>

2. 提交內部審核後的改善單可在 「 審核中 」 查看，如須修改改善單，可點選 「修改內容」 將改善單從內部審核流程退回修改。

\
![](https://content.gitbook.com/content/EqUCL3D5WQfpxJw8NL3P/blobs/lT4sHeZP3qWMN7qgAyBM/%E6%94%B9%E5%96%84%E5%96%AE%E5%9B%9E%E5%A0%B1-12.jpg)  ![](https://content.gitbook.com/content/EqUCL3D5WQfpxJw8NL3P/blobs/iEk8MiWim6SwRQRGwB8P/%E6%94%B9%E5%96%84%E5%96%AE%E5%9B%9E%E5%A0%B1-13.jpg)<br>

* 如不須內部審核，可直接送出改善計畫／改善回報給發文方。

\
![](https://content.gitbook.com/content/EqUCL3D5WQfpxJw8NL3P/blobs/bMDilzF2WPyCo8nJIDzb/IMG_0504.jpg)
