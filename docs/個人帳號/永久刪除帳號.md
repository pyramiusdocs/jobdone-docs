# 永久刪除帳號




!!! danger
    永久刪除帳號僅能由 APP 進行操作，刪除後的帳號無法復原，紀錄也將完全刪除，操作時務必謹慎確認。


***

#### 一、進入 APP 後，點選左上角 點選左上角的齒輪符號 ，按下刪除帳號。

\
![](https://3419978664-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FohH67wtoVrbfIMXiCl2V%2Fuploads%2F6s5XCWBRDdrKZC0QIBh5%2Fimage.png?alt=media\&token=8a20f70e-d959-4869-ab65-d7d74515380e)  ![](https://3419978664-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FohH67wtoVrbfIMXiCl2V%2Fuploads%2FJdHOHfbanLhODLwaBteR%2Fimage.png?alt=media\&token=8db29d30-05b8-4afb-939c-bf6eaab0efcb)<br>

#### 二、按下 「 確定刪除帳號 」 後，系統會要求再次輸入帳號ID與密碼，輸入後點選 「 確定刪除帳號 」 即可。
