# 日報 / 基本資訊


!!! info
    填寫日誌其他內容之前，必須先填寫基本資訊。


## 欄位說明 <a href="#lan-wei-shuo-ming" id="lan-wei-shuo-ming"></a>

點選任一日進入施工日誌詳情，其中開工狀態、上下午天氣：使用者於日誌中自行填寫，其餘欄位項目會從專案資訊自動帶入。

![](images/516e381b2a8b9f76.png)


!!! info
    **資料來源說明**
    
    * 表報編號、開工狀態、上下午天氣：使用者於日誌中自行填寫。
    * 工程名稱、工程核定工期、工期展延天：專案基本資訊。
    * 承攬廠商名稱：專案角色資訊。
    * 填表日期： 自動帶入日誌日期。
    * 累計工期、剩餘工期、實際進度：系統依據所有日報填寫情形智慧計算。
    * 預定計度：可在前往預計進度設定進行設定，如未設定則由系統自動分配。


## 填寫基本資料

點選右上角 「 基本資料 」，填寫開工狀態 ( 必填 )、表報編號、上午天氣、下午天氣、原契約金額、變更後契約金額、契約變更次數等資訊，完成後點選 「 儲存變更 」。
