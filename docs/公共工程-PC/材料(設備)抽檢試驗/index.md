# 材料(設備)抽檢試驗

## 概述

整合材料（設備）檢試驗管制及檢試驗紀錄，並且提供材料（設備）進場品質抽驗紀錄表、材料（設備）查驗申請單與材料試驗暨會驗紀錄表。

***

## 開始使用


!!! info
    在開始使用材料檢試驗功能之前，需先設定專案層級下的「材料管理」。
    
    詳細設定方式請參考「[專案層級 > 材料項目管理](../project_level/materials)」。


### 啟用「材料檢試驗」

由專案管理員，於專案下的基本資料中啟用「材料檢試驗」功能，如下圖。

![](../../images/dc56822f2f0db2d1.png)

開啟此功能後，方可透過功能選單或快速連結進入材料檢試驗頁面。

![](../../images/24d2fce30beb66b9.png)
