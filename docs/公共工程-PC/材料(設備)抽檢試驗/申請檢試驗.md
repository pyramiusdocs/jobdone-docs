# 申請檢試驗

## 檢試驗申請單

1. 進入材料設備管制總表，勾選要申請檢驗的材料項目後，點選 「 ＋申請檢驗 」。
2. 填寫檢試驗申請單內的資訊後，點選 「 儲存 」。

![進入材料設備管制總表，勾選要申請檢驗的材料項目後，點選 「 ＋申請檢驗 」](../../images/9b2f59f607a50db6.png)

![填寫檢試驗申請單內的資訊後，點選 「 儲存 」](../../images/e108dfb9e4ea4a4e.png)

## 列印申請單


!!! info
    為確保列印品質，建議使用 Google 瀏覽器列印。


1. 進入左側的檢試驗申請紀錄，點選指定申請單右側的 「 **⋮** 」，選擇 「 列印申請單 」。
2. 進入預覽介面後，點選 「 列印 」，即可將紙本文件列印提交給監造單位審核。

![](../../images/8f1a70e1bac7fb71.png)

![進入預覽介面後，點選 「 列印 」](../../images/84bcf89872c9f7c0.png)
