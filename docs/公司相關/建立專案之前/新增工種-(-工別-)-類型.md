# 新增工種 ( 工別 ) 類型

## 手動新增 <a href="#materials" id="materials"></a>

### 設定分類（須先完成） <a href="#categories" id="categories"></a>

手動新增須先設定分類，用來管理工種 ( 工別 ) 類型，點擊分類旁的 「 編輯 」 新增分類。

![](images/ec9b855786ddfd3c.png)

### 填寫**工種(工別)類型** <a href="#create" id="create"></a>

點選右上角 「 新增 」 按鈕，點擊 「 新增一筆 」，選擇分類後即可填寫填寫工種(工別)類型。

![](images/31a465c06a5f0ef5.jpg)

***

## **匯入Excel 檔案** <a href="#file_import" id="file_import"></a>

工種 ( 工別 ) 類型可使用指定格式的 Excel 批次匯入，點擊 「 Excel 匯入 」 按鈕開啟檔案匯入功能。


!!! warning
    檔案匯入功能僅可以在沒有任何工種 ( 工別 ) 類型的情況下使用。


![](images/e79c76a63c1775a2.png)

### 下載並匯入 Excel 模板

點選下載 「 工種 ( 工別 ) Excel 模版 」，使用模板填妥資料，上傳檔案後點選 「 匯入 」 即可批次匯入工種 ( 工別 ) 類型資料。

![](images/08b0ac8fddb180a9.png)
