# 臨時工

{% stepper %}
{% step %}
### 安裝 APP

請您下&#x8F09;**「臨時工接單 (Jobdone)」**&#x41;PP，支援 Android 和 iOS 系統。

![](../../images/0bbdc0f65e7eb61b.png)

可掃描下方 QR code

<div>![](../../images/7280e939c697e107.png) ![](../../images/cee79a1d95892994.png)</div>


{% endstep %}

{% step %}
### 註冊

如果您已經擁有帳號，請直接登入。


!!! tip
    每個人只需註冊一個帳號，即可與多家派遣公司進行合作。


填寫您&#x7684;**「帳號 ID」**、**「密碼」**，於閱&#x8B80;**「隱私權聲明」**、**「使用者同意聲明」**、**「服務條款」**&#x540C;意並註冊。

<div>![點選註冊填寫註冊資料](../../images/ffa47193600f06ca.png) ![](../../images/e86498a2c2d3908d.png)</div>

輸入您的手機號碼，&#x4E26;**「接收簡訊驗證碼」**。完成驗證後，填寫您的個人資料，即可完成註冊程序。

<div>![填寫手機號碼&#x26;驗證](../../images/21ad577ddec7f6be.png) ![設定個人檔案&#x26;完成註冊](../../images/cf202f68361aabe8.png)</div>


{% endstep %}

{% step %}
### 登入

登入後，您將看到一個 QR Code，點擊即可放大。

這個 QR Code 是您的個人掃碼檔，可用於快速簽到和簽退（營建商掃碼）。

![](../../images/bf5e9d8f8f3c2022.png)


{% endstep %}

{% step %}
### 與派遣商聯繫

確保派遣商已透&#x904E;**「派遣工管理」**&#x5C07;您加入至派遣工列表內。

詳情請參閱 **➙** 派遣商 - [派遣工管理](lda/lin-shi-gong-guan-li)
{% endstep %}

{% step %}
### 接單與回覆

&#x7576;**「派遣商發出派遣通知」**&#x6642;，App會自動通知您有派工通知，請務必保持App的通知功能開啟。

![](../../images/32e960a427ebe6ae.png)

進&#x5165;**「待回覆派遣單」**，即可看到工單內容。

選擇回覆，決定是否要接單。接受或拒絕後，系統即會回覆派遣商。

<div>![點選派遣通知](../../images/1380438c3e79d4a6.png) ![查看派遣單](../../images/0b01c2f4b43bcda0.png) ![接受/拒絕工作](../../images/f865be01d55b7809.png)</div>


{% endstep %}
{% endstepper %}
