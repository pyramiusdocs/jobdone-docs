# 建立公司專案


!!! warning
    需要先開啟[**功能授權**](gong-neng-shou-quan)才可以建立公司專案。


登入帳號後，在左側的選單中選擇  「 專案清單 」，點選 「 ＋新增專案 」，依序填寫：

* 一般建案或公共工程
* 專案資料
* 監造或營造身份
* 選擇專案經理

即可建立新的專案。
