# 註冊

進入 Jobdone 的網頁點選 「 註冊 」 ，設定想要的帳號密碼後，輸入手機號碼接收驗證碼，設定個人資料後即可註冊完成。


!!! info
    若無法收到驗證簡訊，請關閉 Whoscall 等類型的的阻擋軟體再嘗試。
