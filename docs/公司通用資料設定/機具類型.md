# 機具類型

系統提&#x4F9B;**「手動新增」**&#x53CA;**「Excel匯入」**&#x5169;種方式編列您的機具類型資料。

此功能用於設定後續案場實際會用到的施工機具清單（例如：挖掘機、吊車、發電機等）。

* 預設機制：在公司層級設定好機具類型後，系統會自動將這些資料預設為各專案內部的機具清單。
* 專案彈性：雖然各案場會自動套用公司層級的資料，但進入特定專案後，仍可根據該案場的實際需求進行個別調整。


!!! warning
    &#x65BC;**「施工日誌」**&#x6240;選用之機具，都將以此處資料為依據，務必確認您的機具資料已妥善填寫。


![](../images/018af7fc9c015641.png)

## 01｜手動新增

點擊主頁面&#x4E4B;**「公司通用資料設定」**&#x5167;&#x7684;**「機具類型」**，開始編輯機具資料。

請根據以下流程操作：

{% stepper %}
{% step %}
### 進行分類設定

點擊下圖分類旁&#x4E4B;**「編輯」**，開始新增您的分類。

![](../images/ca8c4a5ac0014032.png)

於建立施工機具前，務必先建立機具分類，請依照以下步驟進行：

1. 開啟編輯：進入機具類型管理畫面，點選下方的 ![](https://2002560121-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEqUCL3D5WQfpxJw8NL3P%2Fuploads%2FW2xYMl3m497owt77TINW%2Fimage.png?alt=media\&token=c6f2c182-ea93-49c7-b4ac-6ba276b29321) 圖示，即可新增空白欄位。
2. 填寫資訊：在產生的空白欄位中，填入機具分類名稱。
3. 完成儲存：確認名稱正確後，點選 ![](https://2002560121-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEqUCL3D5WQfpxJw8NL3P%2Fuploads%2F0RPGm4h3GLfUXu2LDXNw%2Fimage.png?alt=media\&token=6400c5e4-a908-4bcf-98ba-d0a3680b12bb) 鍵即可完成設定。

![圖二 - 填寫機具分類/儲存](../images/de82fa613b86ea1a.png)
{% endstep %}

![圖四 - 填寫機具資料](../images/9148f33494354a86.png)

### 編輯機具資料

當專案機具清單建立完成後，系統仍保有極高的調整彈性，以應對不同施工階段的變動需求：

若後續需調整機具資訊，請點選頁面中的 ![](https://2002560121-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEqUCL3D5WQfpxJw8NL3P%2Fuploads%2FOZpNY6ujeCIlNauy3LxB%2Fimage.png?alt=media\&token=c564ce53-b138-45a1-9439-9070477ae4f7) 按鈕。進入編輯模式後，您可以針對現有資料進行以下操作：

{% endtab %}

{% tab title="刪除機具資料" %}
移除該專案已不再使用的機具項目，保持選單精簡。
{% endtab %}
{% endtabs %}

![](../images/06b7d9399df3739c.png)

![](../images/5789ea632b4010ac.png)

## 02｜Excel 匯入

點擊主頁面&#x4E4B;**「公司通用資料設定」**&#x5167;&#x7684;**「機具類型」**，開始編輯機具資料。


!!! warning
    Excel 匯入功能僅能在尚未新增任何機具類型資料時使用，匯入後則無法再匯入。
    
    透過Excel匯入後，若您需要更動/增加機具資料，則需透過手動編輯。
    
    由於檔案僅能上傳一次，若您需要重新匯入Excel資料，則需先將原有資料全部刪除。
    
    ( 刪除 Jobdone 公司通用資料設定之工種資料，而非Excel。)


請根據以下流程操作：

{% stepper %}
{% step %}
### 下載 Excel 模板

點擊(圖一)紅框圈選處之「Excel匯入」，進入(圖二)頁面後，開始下載Excel機具類型模板。

![機具類型 - 模板下載 - 圖一](../images/2315dc58f603b824.png)

開啟 Excel 機具模板 視窗後，於畫面中的『Excel 模板下載』欄位，點選 ![](https://2002560121-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEqUCL3D5WQfpxJw8NL3P%2Fuploads%2Ftnmjy0HdIjItIiObxNJK%2Fimage.png?alt=media\&token=7b58d077-fbf3-426f-b056-dc7b04269bf6) 圖示，即可將標準範本儲存至您的電腦。

1. 填寫資料：開啟下載的 Excel 檔案，依據格式填入案場預計使用的機具清單。

![機具類型 - 模板下載 - 圖二](../images/fea22d07f3101d01.png)

檔案畫面如下所示，依據模板表格填&#x5BEB;**「機具分類」**&#x8207;**「工種類型名稱」**。

![](../images/fb08b9c8798915f7.png)
{% endstep %}

{% step %}
### 填寫 Excel 模板


!!! warning

    ( 刪除 Jobdone 公司通用資料設定之機具資料，而非Excel )。
