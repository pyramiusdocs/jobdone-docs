# 公司自主檢查範本

匯入範本公司檢查範本分為兩種類型：**條列型範本**與**檔案型範本**。

{% tabs %}
{% tab title="條例型範本" %}
可由使用者自行新增欄位，自由編列檢查項目與檢查標準，適用於需要彈性調整的檢核內容。
{% endtab %}

{% tab title="檔案型範本" %}
可上傳既有的檢查表格檔案，提供公司其他成員日後下載使用，適用於標準化或固定格式的表單。
{% endtab %}
{% endtabs %}


!!! warning
    請注意：
    
    建立範本前，須先完成「分項工程」及「專案職稱」之設定，以利後續範本項目與檢查人員角色的正確對應與指派。
    
    有關<kbd>**分項工程**</kbd>與<kbd>**專案職稱設定**</kbd>之詳細操作流程，請參閱 ➙ [project-role](project-role "mention") & [subdivision](subdivision "mention")

## 02｜匯入/瀏覽檢查範本 (系統參考例)

符合工程會（公共工程委員會）對公共工程品質管理的要求，Jobdone 提供多達 200 種以上的自主檢查表範例，涵蓋各類常見工項與檢查標準。

您可直接從範本庫中匯入所需的檢查範本，快速建立專案檢查流程，省去逐項手動新增的時間與繁瑣作業。匯入後，仍可依據實際施工內容進行調整與編輯，確保符合現場需求與工程規範。


!!! info
    範例內容僅供參考示意，實際檢查項目及標準請依各專案需求自行編輯與調整，以符合實務作業及檢核要求。


<div>![圖一 - 點選「建立範本」](../images/ac9e8b30900c8102.png) ![圖二 - 開啟功能選單](../images/4146d159f10a5741.png)</div>

### 02 - 1｜匯入檢查範本

如圖三所示，開啟功能選單後，請點選<kbd>**匯入檢查範本 (系統參考例)**</kbd>。

![](../images/04c0f11284572a7b.png)

開啟範本篩選器後，您可依據檢查範本的**編號**、**名稱**、**類型**及**標籤**進行篩選，快速找出符合您需求的範本。

![圖四 - 篩選所需範本](../images/935bb575c05da066.png)

篩選完畢後，即可從符合條件的範本列表中**選擇**所需範本進行匯入，並將所選範本歸類至指定的分項工程。

![圖五 - 選擇/匯入範本](../images/1d9c51eb24dc7e40.png)

### 02 - 2｜瀏覽檢查範本

如圖六所示，開啟功能選單後，請點選<kbd>**瀏覽檢查範本(系統參考例)內容**</kbd>。

即可預覽系統提供的範本詳細項目與檢查標準，供您參考評估是否匯入使用。

![](../images/456f4d27809252d4.png)

## 03｜檢查項目 & 分類

![圖一 - 檢查範本內部說明](../images/9ca7663d701c111e.png)

### 03 - 1｜新增檢查分類

如圖二 \~ 圖三所示，進入範本內部後，於檢查項目分類欄位右側點&#x9078;**「管理」**，即可開啟檢查分類管理頁面，進行分類的新增、編輯或刪除操作。

<div>![圖二 - 點選「管理」](../images/204107314a567a5c.png) ![圖三 - 開啟分類管理頁面](../images/cd4e793002259e4b.png)</div>

如圖四 \~ 圖五所示，於新增欄位輸入分類名稱後，點&#x9078;**「+新增」**，該分類即會顯示於分類列表中，之後即可針對該分類編列對應的檢查項目。

<div>![圖四 - 填寫檢查分類/新增](../images/88f44a102fbaa1ab.png) ![圖五 - 完成畫面](../images/8de03918f283ecfc.png)</div>

### 03 - 2｜編輯/刪除檢查分類

如圖六、圖七所示，您可點&#x9078;**「**![](../images/ef64fcab71ea8e5a.png)**」**&#x7DE8;輯檢查分類名稱，或點&#x9078;**「**![](../images/6a5438166f414067.png)」刪除不需要的檢查分類。


!!! warning
    若欲刪除之檢查分類下仍有檢查項目，系統將提供讓您選擇：<kbd>**是否一併刪除該分類下所有檢查項目**</kbd>，或將這些檢查項目自動移轉至「未分類」中。請依實際需求謹慎操作。


<div>![圖六 - 編輯檢查分類](../images/c6eb8e3d37ac91fb.png) ![圖七 - 刪除檢查分類](../images/8ed220537d579e73.png)</div>
