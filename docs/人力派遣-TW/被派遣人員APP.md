# 被派遣人員APP



## 安裝

請派遣工下載Android或iOS的APP。可直接掃描QR Code或至「Google Play」、「App Store」搜尋 【臨時工接單(Jobdone)】。

<div>![](../images/ccdfdd1b97e69cd1.png) ![](../images/510de1ea19223f66.png)</div>



## 註冊

每個人只需要註冊一個帳號，即可與多家仲介公司合作。若已經有帳號，直接登入不需要再註冊。

1. 若沒有註冊過，請點選註冊。簡單設定 「帳號ID」、「密碼」。閱讀 「隱私權聲明」、「使用者同意聲明」、「服務條款」後勾選 【同意】。

<div align="left">![](../images/7d961f6626957cf6.jpg) ![](../images/b0aadbfbfe161bd5.jpg)</div>



2. 請輸入手機號碼，並收取簡訊驗證碼。完成驗證後，填入您的個人基本資料，即可完成註冊程序。

<div>![](../images/3a26e6f12d8c694b.jpg) ![](../images/471649b04fbd3649.jpg)</div>



3. 登入後，會在中間看到一個QR Code，點擊可以放大。該QR Code就是個人的掃碼檔，可用在快速簽到、簽退。

![](../images/9e07f95952b49585.png)



## 接單與回覆



裝好APP並且註冊完成後，只要等著派遣工作上門。當派遣商發出派遣通知時，手機APP會自動發出訊息，請務必保持APP的通知功能開啟。

<div align="left">![](../images/9fc335566a5679b0.png) ![](../images/5f55082e2066b46c.jpg)</div>

**進入待回覆的派遣單，可以看到工單內容。**

<div align="left">![](../images/5ff42e7ee1e1e158.jpg) ![](../images/387aba88e996660f.jpg)</div>

**選擇回覆，決定是否要接單。**

![](../images/f3108815fceeb881.jpg)

**接受或拒絕後，系統即會回覆派遣商。**
