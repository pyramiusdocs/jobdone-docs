# 派遣工管理

## 步驟

1. 邀請合作的派遣工下載並註冊 【臨時工接單(Jobdone)】 APP。
2. 在【派遣工管理】中關聯該派遣工的帳號。
3. 建立人員相關資料。


!!! warning
    請注意順序，要請派遣工先註冊帳號，才能進行關聯。




## 臨時工接單(Jobdone) APP

派遣工註冊請參閱 [「被派遣人員APP」](bei-pai-qian-ren-yuan-app)

<div align="left">![](../images/a687f3c7966b4900.png)</div>

## 新增派遣人力

在派遣工管理，點擊新增派遣人力。輸入派遣工已註冊的手機號碼進行關聯。

![](../images/07b97151382aa1c4.png)

填寫派遣工個人資料，完成派遣工登記。


!!! warning
    請注意，這裏的資料只會紀錄在各別仲介公司下，相關資料不會傳送至APP，或與任何其它公司分享。


![](../images/0a6471089c9f3a47.png)
