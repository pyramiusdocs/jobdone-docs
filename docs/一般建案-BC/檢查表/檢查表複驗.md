# 檢查表複驗

## 網頁版

### 複驗

進入檢查表後，點選 「 不合格 」 的檢查結果，右方檢查紀錄會顯示 「 + 複驗紀錄 」 或 「 發起改善 」 按鈕，點擊按鈕後即可填寫複驗紀錄或發起改善單。

![](images/b7a126e682effeac.png)

### 填寫複驗紀錄

點選 「 + 複驗紀錄 」 即可填寫複驗紀錄，填寫完畢後點選 「 儲存複驗結果 」。


!!! info
    在[**列印檢查報告**](print)時，若選擇 「 交付模式 」 則會將複驗紀錄取代不合格紀錄，並標記為合格。


![](images/5965f980b10696ad.png)

### 發起改善單

針對不合格缺失，可以發起改善單追蹤改善情況，點選 「 發起改善 」 ，即可[**發起改善單**](../../../project_level/corrective/build#tou-guo-gong-neng-fa-qi-gai-shan-dan)。


!!! info
    若建立不合格紀錄時有指定缺失責任人，此處會自動帶入缺失責任人作為改善單的受文者。若想選擇其他受文者也可以點選 「 重新選擇 」 變更受文者。


![](images/3b2ba35cd8794054.png)

## APP

### 複驗
