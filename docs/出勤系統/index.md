# 出勤系統

### 01｜出勤系統概覽

這套出勤系統是為 Jobdone 平台上所有使用者設計的新功能，無論是臨時工派遣、營建管理還是施工製造模組的使用者，都能夠享受到精確、便捷的出勤管理服務。系統提供了靈活的功能，滿足不同工作類型的需求，讓用戶無論在哪個領域，都能高效管理出勤、加班、休假及相關流程。

在現代化的專案管理中，出勤系統不只是『記錄上下班時間』，它是人力資源調度與成本控管的核心數據。系統整合了 App 端的 GPS 定位精確性 與 Web 端的 行政管理便利性，讓每一筆打卡、每一小時的加班，都能轉化為透明、具備可追溯性的數位憑證，確保勞資雙方的權益皆受到保障。

***

### 02｜功能簡介

出勤系統的核心在於「權限分立」與「數據實證」，透過結構化的管理介面，滿足從現場人員到人資管理者的多樣化需求：

1. **多層次權限管理 (Permission Control)：** 系統特別抽離出「人資權限」，使得管理層與普通成員的操作界限明確。人資管理員具備最高審核權，能針對成員的休假、加班申請進行核定，並可針對不同員工彈性調整假別時數，確保福利制度與排班計畫能完美對接。
2. **打卡與 GPS 定位 (Attendance & Geofencing)：**&#x900F;過 App 端打卡時，系統會自動擷取當下的具體地理位置（經緯度）。在分散的施工現場或多點作業環境下，這些 GPS 數據能即時驗證員工是否確實到達工作崗位，有效杜絕虛報出勤，解決內部同仁因多點辦公導致的考勤管理難點。
3. **加班與休假申請 (OT & Leave Request)：**&#x5167;部成員可隨時於系統內提交加班或休假申請，並依照公司定義的流程送交人資審核。所有的申請時數均由系統自動統計，人資可依據實務狀況進行個別微調，確保所有數據均符合公司內部規範，避免人工統計產生的誤差。
4. **Excel 匯出功能 (Data Export)：**&#x4EBA;資管理員能將全體內部成員的打卡、加班、休假紀錄一鍵匯出為 Excel 文件。這不僅讓公司內部的出勤數據更加透明、易於保存，更大幅提升了後端財務或行政人員在處理月底薪資對帳時的作業效率。

!!! info
    #### 💡 價值
    
    1. 透過數位化的申請與審核軌跡，減少內部公文往返時間，讓管理層與員工之間的出勤資訊同步透明。
    2. 將內部成員的工時數據與案場地點結合，幫助公司更精確地掌握專案的人力支出成本。
    
    這套出勤系統是企業內部管理的數位基石，透過嚴謹的流程與數據紀錄，為公司建立一套專業、公平且具備數位化管理效能的出勤體系。
