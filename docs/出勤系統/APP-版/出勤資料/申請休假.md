# 申請休假

進入出勤系統主頁面後，如圖一紅框圈選處，點選右上角&#x4E4B;**「選單」**，即可見圖二畫面並點&#x9078;**「申請休假」**。

即可進入圖三畫面開始填寫休假申請資料（包括：**休假時間**、**休假時數**、**假別**與**休假原因**）。


!!! tip
    個人申請的休假紀錄，都可於個人紀錄頁面的<kbd>**休假紀錄**</kbd>頁籤查看。


<div>![圖一 - 點選「選單」](../../../images/958c0c56e09761f2.png) ![圖二 - 點選「申請休假」](../../../images/42be51bb499e739b.png) ![圖三 - 輸入休假申請資料 ](../../../images/2daaeda6b447368b.png)</div>

#### 填寫休假時間

點選圖四之紅框圈選處，分別填寫**休假開始時間**與**結束時間**。

點選圖四紅框圈選處 **➙** 進入圖五頁面選取**休假日期 ➙** 進入圖六畫面選取**休假申請時間。**

<div>![圖四 - 點選時間欄位](../../../images/6716b82fe5d26ee5.png) ![圖五 - 選取休假日期](../../../images/f3c3c70e6b5d2a4e.png) ![圖六 - 選取休假時間](../../../images/04953f3bee453847.png)</div>

#### 填寫假別

點選圖七之紅框圈選處，開始選擇假別(圖八)，選取完畢畫面即如圖九。

<div>![圖七 - 點選假別欄位](../../../images/67f30584dcbc78bd.png) ![圖八 - 選擇假別](../../../images/1929ab638bdda473.png) ![圖九 - 完成畫面](../../../images/6d9a9b0b79b6d2f0.png)</div>
