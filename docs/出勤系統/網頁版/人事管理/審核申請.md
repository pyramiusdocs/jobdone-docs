# 審核申請

當有公司成員提出加班/休假申請時，人資成員可於此處審核該加班/休假申請。


!!! warning
    請注意，僅具有人資權限之成員可審核全公司成員之加班&休假申請。


![審核申請畫面](../../../images/346affa862f2a94b.png)

***

## 💪 01｜加班申請

如下點選圖紅框圈選處，進入出勤系統之審核申請頁面後，點選<kbd>**加班申請**</kbd>頁籤。並會於頁籤旁顯示共有幾筆申請紀錄尚未審核。

此頁面會詳細顯示所有加班申請資訊，包括：**申請人**、**加班開始/結束時間**、**加班時數**及**核准狀態**等。

![點選加班申請頁籤](../../../images/3b1498e0caa4fce0.png)

選取<kbd>**加班申請**</kbd>頁籤後，如左圖紅框圈選處，於欲審核之申請右側操作欄位，點&#x9078;**「⋮」**&#x4E4B;**「審核加班申請」**。

即可查看該**成員名稱**、**加班時間**、**加班時數**、**所對應之專案名稱**及**加班原因**。

您必須填妥審核備註後，再決定是否拒絕核准/核准申請。

<div>![左圖 - 點選審核加班申請](../../../images/0e705a0880cadc08.png) ![右圖 - 查看申請資訊＆審核申請](../../../images/ab728625a00fd3bf.png)</div>

***

### 01 - 1｜篩選申請資料

如下圖紅框圈選處，您可於此處透過**狀態**、**加班日期**及**成員名稱**篩選加班申請資料。

![篩選欄位](../../../images/9d6a25afdb0659dc.png)

點選左圖紅框圈選處，開啟選單選擇欲篩選的狀態。包括：**待審核**、**已核准**及**拒絕核准** (右圖)。

{% endtab %}

{% endtabs %}

#### 日期篩選

點選左圖紅框圈選處，開啟月曆表選擇特定日期，以篩選該日期的所有加班申請。
