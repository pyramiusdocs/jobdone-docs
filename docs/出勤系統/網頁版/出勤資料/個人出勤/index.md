# 個人出勤

使用者可於此處查看您個人的出勤資料，包括：**打卡記錄**、**加班紀錄**、**休假紀錄**等。

![個人出勤頁面](../../../../images/4692a205d676d0f0.png)

***

## 🕒 01｜打卡記錄

如圖一紅框圈選處，進入出勤系統之個人出勤頁面後，點選<kbd>**打卡記錄**</kbd>頁籤。

即可查看您一個月內的所有打卡記錄，包括：**上班時間**、**下班時間**及**打卡位置**。


!!! info
    紀錄顯示以一個月為單位呈現。


![圖一 - 點選打卡記錄頁籤](../../../../images/8b5cfa1dc81658db.png)

### 01 - 1｜篩選日期

如圖二紅框圈選處，您可調整日期至欲查看的月份，點&#x9078;**「**![](../../../../images/c1ad9eded886c6ab.png)**」**&#x7BAD;頭符號與點&#x9078;**「**![](../../../../images/7edcbbc65b591fca.png)**」**&#x65E5;曆符號有不同作用：

{% tabs %}
{% tab title="點選箭頭符號" %}
以一個月為單位，可向前選擇至上個月或向後選擇至下個月。
{% endtab %}

{% tab title="點選日曆符號" %}
可開啟選擇欄位，快速選擇所需的年份與月份。
{% endtab %}
{% endtabs %}

![圖二 - 日期篩選](../../../../images/a1fa29b6bccb8b6d.png)

#### 點選 ![](../../../../images/18cbb6aa18016c80.png)

點選日曆符號(左圖)後，即可進入右圖開始選擇所需的年份與月份。

<div>![左圖 - 點擊日曆符號](../../../../images/e6857e284a5a6d3b.png) ![右圖 - 選擇年份與月份](../../../../images/ce7567e2b46a7480.png)</div>

***

### 01 - 2｜查看打卡位置

如下圖紅框圈選處，於打卡位置欄位，點選下方的經緯度資料，即可進入**Google 地圖**詳細查看位置。


!!! tip
    使用App進行打卡時，系統即會自動依據您的所在位置儲存經緯度資料。


![查看打卡位置](../../../../images/28d37ce2568dea9f.png)

***

## 🔥 02｜加班記錄

如圖一紅框圈選處，進入出勤系統之個人出勤頁面後，點選<kbd>**加班紀錄**</kbd>頁籤。

即可查看您一個月內的所有加班記錄，包括：**加班開始/結束時間**、**加班時數**及**核准狀態**。


!!! info
    紀錄顯示以一個月為單位呈現。


![圖一 - 點選加班紀錄頁籤](../../../../images/6a27021a998d68c2.png)

***

### 02 - 1｜核准狀態說明

共有**審核中**、**已核准**及**拒絕核准**三種狀態，可於各加班紀錄之核准狀態欄位查看。


!!! warning
    僅具有<kbd>**人資**</kbd>權限之公司成員，才能審核加班申請。


{% tabs %}
{% tab title="審核中" %}
人資成員**尚未審核**加班申請。
{% endtab %}

{% tab title="已核准" %}
人資成員已審核並**核准**該加班申請。
{% endtab %}

{% tab title="拒絕核准" %}
人資成員已審核並**拒絕**該加班申請。
{% endtab %}
{% endtabs %}

![加班紀錄 - 核准狀態](../../../../images/f1eabcb6545a23ff.png)

***

### 02 - 2｜查看與撤回申請

您可於各加班申請紀錄之**操作**欄位，**查看**您的加班紀錄或**撤回**您的加班紀錄。

#### 查看加班紀錄

如左圖紅框圈選處，於欲查看的加班紀錄右側操作欄位，點&#x9078;**「**&#xD83D;&#xDD0D;**」**&#x5373;可進入右圖畫面查看加班資訊。

如右圖，即可查看**加班原因**、加班所對應的**專案名稱**及**審核備註**。

<div>![左圖 - 點選放大鏡符號](../../../../images/b35433a254af74f1.png) ![右圖 - 查看加班資訊](../../../../images/695eefdaa446d8f1.png)</div>

#### 撤回加班申請


!!! warning
    僅**審核中**的加班紀錄可以撤回申請。


如下圖紅框圈選處，於欲撤回的加班紀錄右側操作欄位，點&#x9078;**「⋮」**&#x5373;可選&#x64C7;**「撤回加班申請」**。

![撤回加班申請](../../../../images/cefbfe0e5c3564e6.png)
