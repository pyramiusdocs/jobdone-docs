# 釋疑單

登入帳號後，在系統畫面中選擇 「 我的釋疑單 」 進入釋疑單介面。

![](images/08d4e1434d00dbbe.png)

## 發起釋疑單

選擇右上角 「 ＋發起釋疑單 」 ，填寫釋疑單內容欄位，點選 「 選擇指派對象 」，選擇收單人後點選 「 確認發單 」 即可送出釋疑單。


!!! warning
    需先將收單人[**加入聯絡人**](qian-zhi-zuo-ye/jia-ru-lian-luo-ren)，才能在釋疑單中選取。

![填寫釋疑單內容欄位，點選 「 選擇指派對象 」](images/d5638971141f7af4.png)

![選擇收單人後點選 「 確認發單 」 即可送出釋疑單](images/ebab53fc481d65bc.png)

## 專案釋疑單

如果要將改善單歸類在公司專案下，可在填寫欄位時選擇 「 公司專案 」。


!!! warning
    需要先[**建立公司專案**](qian-zhi-zuo-ye/jian-li-gong-si-zhuan-an)後，才可以在釋疑單中選取。


![](images/47cf54fe3e82d259.png)

## 查看釋疑單

進入改善單介面後，即可看見釋疑單的回報情形，點擊指定釋疑單可查閱更詳細的內容。
