# 註冊

進入 Jobdone 的網頁點選 「 註冊 」 ，設定想要的帳號密碼後，輸入手機號碼接收驗證碼，設定個人資料後即可註冊完成。


!!! info
    若無法收到驗證簡訊，請關閉 Whoscall 等類型的的阻擋軟體再嘗試。

![輸入手機號碼後，點選 「 驗證手機 」](images/d0a66aba7f8e945a.png)

![手機會收到一組簡訊驗證碼，輸入後點選 「 下一步 」](images/4de5d77e6d9387f1.png)

![設定個人資料後即可註冊完成](images/58139b17be4cd84c.png)


!!! warning
    提醒您：請務必填寫有效的**個人信箱**，忘記密碼才能重新設定。
