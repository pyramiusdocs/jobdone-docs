# 加入聯絡人

登入帳號後，在左上角的選單總表中選擇 「 我的聯絡人 」，點選 「 ＋新增聯絡人 」，輸入聯絡人的電話號碼後搜尋 「 加入聯絡人 」。


!!! warning
    聯絡人需使用電話號碼註冊 Jobdone 帳號後才能被搜尋。
