# 派工單

登入帳號後，在系統畫面中選擇 「 我的派工單 」 進入派工單介面。

![](images/0dcd7de249dcd554.png)

## 發起一般派工單

選擇右上角的 「 ＋發起派工單 」，填寫派工單內容欄位，點選 「 選擇指派對象 」，選擇派工對象後即可送出工單。


!!! warning
    需先將要指派的對象[**加入聯絡人**](qian-zhi-zuo-ye/jia-ru-lian-luo-ren)，才能在派工單中選取。

## 發起專案派工單

如果要將派工單歸類在公司專案下，可在填寫欄位時選擇 「 公司專案 」。


!!! warning
    需要先[**建立公司專案**](qian-zhi-zuo-ye/jian-li-gong-si-zhuan-an)後，才可以在派工單中選取。


![](images/ffc527bee0d299b4.png)

## 查看派工單回報

在我的派工單中選擇 「 由我發單 」 介面，即可看到該派工單的回報情況。

![](images/2a96f787121d7b30.png)
