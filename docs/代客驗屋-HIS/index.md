# 代客驗屋 HIS

Jobdone代客驗屋是專為**科技驗屋公司**所設計一套簡便的工具，可以幫助驗屋公司縮短撰寫驗屋報告的文書處理時間，原則上在現場兩個小時的驗屋過程，使用Jobdone代客驗屋APP邊做檢測、邊拍照做紀錄的同時，就已經將驗屋報告彙整好了，並且上傳至Jobdone雲端系統，自動套表列印驗屋報告，也可以讓屋主線上閱覽動態的驗屋報告。

此外，Jobdone代客驗屋也可以跟建設公司的驗收單直接資料整合，節省去雙方重新輸入資料的時間，讓缺失改善的效率提升，縮短屋主驗收交屋的時間。
