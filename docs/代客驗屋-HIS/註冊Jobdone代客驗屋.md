# 註冊Jobdone代客驗屋

![](../images/614b971a575eaab4.png)

{% stepper %}
{% step %}
#### 下載：安裝Jobdone代客驗屋APP

<div align="left">![](../images/e98b8447319e12c2.png)</div>
{% endstep %}

{% step %}
#### 註冊：按下「註冊」按鈕註冊帳號

<div align="left">![](../images/86441c24676b3367.png)</div>
{% endstep %}

{% step %}
#### 註冊ID、密碼、勾選並同意隱私聲明、按下「註冊」按鈕。

<div align="left">![](../images/f97a17ef40832c47.png)</div>
{% endstep %}

{% step %}
#### 輸入您的手機號碼，按下「驗證手機」按鈕。

<div align="left">![](../images/7cbe253ae581f6b5.png)</div>
{% endstep %}

{% step %}
#### 驗證通過就會看到Jobdone代客驗屋APP首頁，表示註冊成功。

<div align="left">![](../images/3bc0919bd1b673df.png)</div>
{% endstep %}

{% step %}
#### 此時，Jobdone代客驗屋APP應該沒有資料可以使用。所以我們需要登入到Jobdone網站，進行驗屋工作的設定。請使用「電腦的瀏覽器」連結到 https://www.jobdone.cc&#x20;

#### 點選右上角的『註冊/登入Jobdone』按鈕。
{% endstep %}

{% step %}
![](../images/61ce0a82f0a08f26.png)


{% endstep %}

{% step %}
#### 將剛才註冊的帳號及密碼登入Jobdone系統，按下「登入」按鈕。

![](../images/29bdcfc483b4eeb2.png)
{% endstep %}

{% step %}
#### 登入進來需要先建立公司，請選擇『建立公司，申請試用』按鈕。

![](../images/9ebb9339be4ceb41.png)

#### 選擇公司適用的模組，請選擇『代客驗屋』，按下建立公司按鈕。

![](../images/0e03f6c00097829d.png)

#### 公司建立完成會出現以下畫面，請按下『申請免費試用』

![](../images/e804ac476bd5bd19.png)

#### 此時會提供30天的免費試用期

![](../images/66c391753c2f35fc.png)

#### 接下來會看到免費試用的期限，請點選左上角的『九個點』，選擇『Jobdone客驗屋』

![](../images/afbcc55966a0260b.png)

#### 就會看到以下操作畫面，下一個章節將介紹系統設定。

![](../images/72112f6462280e40.png)
{% endstep %}
{% endstepper %}
