# 釋疑單

### ♨ 當你進場發現實際狀況與圖面不合

### ♨ 當你發現業主的要求無法執行

### ♨ 當你作業到一半發現介面衝突或干涉



過去你會去找到工地負責人，口頭詢問，找不到可能這個工班就回報公司，然後就停擺了。

現在Jobdone有更好的選擇，每個人都可以當下就把遇到的狀況紀錄下來，可以是文字、錄影、拍照，然後即時發出詢問單。如果不確定誰能回答，沒關係，這一張單可以被接收方轉發，一直到找到正確的人。

這樣的好處是，遇到問題的當下，已經清楚的紀錄了問題的狀況、原因、現場的情形，最重要的是發問的時間。所有的作業都有憑有據，可以被追蹤，就不容易發生有人提問，但放在LINE或者口頭說過，之後就忘了處理。
