# 調度中心

### 一、功能概述

**調度中心**為 Jobdone 系統中「施工製造模組」的核心管理平台，提供**公司層級**的施工資源統籌與調度功能。其設計目的是協助管理者從全公司視角進行跨專案、跨人力、跨設備的即時調度與監控，確保施工效率與資源配置最佳化。


!!! tip
    透過調度中心，Jobdone 將企業施工調度數位化、可視化，協助公司從「人、車、工」三面向達成全面協同管理，打造高效率的工程執行體系。
