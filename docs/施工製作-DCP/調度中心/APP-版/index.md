# APP 版

## 01｜如何進入調度中心？

{% stepper %}
{% step %}
### 使用 Jobdone App

請您下載 **Jobdone** App版，支援 Android 和 iOS 系統。&#x20;

<div>![](../../../images/52f1155396042c2b.png) ![](../../../images/5584cf55eff27050.png)</div>

<div>![](../../../images/9e20fd2f455e680b.png) ![](../../../images/de4ca9b000332026.png)</div>
{% endstep %}

{% step %}
### 進入調度中心功能

進入 App 系統後，點選下方&#x7684;**「公司通用功能」**，再選擇<kbd>**調度中心**</kbd>，即可進入並使用相關功能。


!!! warning
    請注意，務必確認所登入之帳號使&#x7528;**「施工製造」**&#x6A21;組。


![圖一 - 調度中心操作流程](../../../images/9cdec80e598901d1.png)
{% endstep %}

{% step %}
### 切換頁面 (派工單/派車單)

如下圖所示，您可依需求查看所需資料，點選相應頁籤即可切換至<kbd><mark style="color:purple;">派工單<mark style="color:purple;"></kbd>或<kbd><mark style="color:purple;">派車單<mark style="color:purple;"></kbd>頁面。

![圖二 - 切換頁面說明 (派工單/派車單)](../../../images/f4b7a874bfd37fd0.png)
{% endstep %}
{% endstepper %}
