# 資產設備盤點總覽

![](../../images/977093be6fdcdb13.png)

**資產設備盤點總覽 (網頁端)**\
本頁面彙整所有於「資產設備管理」中所建立的資產資料，提供公司完整的資產清單與狀態總覽。您可透過此頁面執行以下功能：

* **資產狀態管理**：查看每筆資產目前的可用狀態 (正常使用/保養中/維修中/報廢等)，並可進行狀態調整。
* **盤點紀錄查詢**：查閱每筆資產的歷次盤點記錄、盤點人員與時間，掌握資產使用與管理履歷。
* **QR Code 列印與盤點準備**：系統提供專屬 QR Code，可批次產出、列印張貼，方便現場透過 Jobdone App 掃描進行快速盤點。

夠過此總覽頁面，管理者可即時掌握資產分布與使用狀況，提升設備管理效率與盤點正確性。

![](../../images/4ff2f5ed1816243f.png)

***

## 01｜盤點

**網頁上的盤點**，主要用於設定與準備盤點作業。在 Jobdone 系統中，您可透過網頁端選取欲盤點的資產，系統會自動為每項資產產出對應的專屬 QR Code。這些 QR Code 可列印並張貼於各資產實體上，作為識別標籤。日後現場盤點時，只需使用手機 App 掃描該 QR Code，即可快速完成資產盤點、位置確認與狀態更新，大幅提升盤點效率與準確性。

{% stepper %}
{% step %}
### 選取資產 (欲盤點)

進入主頁面後，點選右上方的<kbd><mark style="color:purple;">**批次動作**<mark style="color:purple;"></kbd>，即可開始選取欲盤點之資產設備，進行後續的盤點操作。

![](../../images/e9c1bab24347adc4.png)

您可選擇以一次勾選 (全選) 所有資產設備，或逐一勾選欲盤點的項目，依需求靈活操作。

如需取消盤點作業，請點選右上方的<kbd>**退出批次動作**</kbd>，即可結束本次批次選取流程。

![圖二 - 勾選資產/退出批次動作](../../images/c883bdf66f59f50f.png)

如圖三所示，將欲盤點之資產勾選完畢後，點選<kbd><mark style="color:purple;">**產生QRCode**<mark style="color:purple;"></kbd>即可為所選資產生成對應的 QR 碼，供列印張貼及後續掃描盤點使用。

![圖三 - 產生QR Code](../../images/c75222b832eb850e.png)
{% endstep %}

## 02｜變更資產位置

**變更資產位置**功能，主要搭配「批次動作」使用，讓使用者可一次選取多筆資產，統一調整其「所在位置」。此功能適用於資產設備移動至新案場或倉儲地點時，便於即時更新資產資訊，確保資產管理正確無誤。

{% stepper %}
{% step %}
### 選取資產

進入主頁面後，點選右上方的<kbd><mark style="color:purple;">**批次動作**<mark style="color:purple;"></kbd>，即可開始選取欲變更位置之資產，進行後續操作。

![](../../images/9c8d505658b9ae57.png)

您可選擇以一次勾選 (全選) 所有資產設備，或逐一勾選欲變更位置的項目，依需求靈活操作。

如需取消變更作業，請點選右上方的<kbd>**退出批次動作**</kbd>，即可結束本次批次選取流程。

![圖二 - 勾選資產/退出批次動作](../../images/92f44876302fb0e5.png)

如圖三所示，選取欲變更之資產並確認無誤後，請點選 ![](../../images/70d530cc3268a2f9.png)，系統將開啟操作視窗，您可於其中選擇新的所在專案位置，完成位置異動設定。

![圖三 - 變更資產位置](../../images/34b873d21da65ceb.png)
{% endstep %}

{% step %}
### 變更資產位置

開啟選單後，系統將顯示專案清單內之所有專案，您可於其中選取欲變更的資產新位置 (專案)，以完成位置調整設定。

![圖四 - 變更資產位置操作說明](../../images/777f460ea82877b2.png)

選取完畢並確認無誤後，請點&#x9078;**「儲存」**。即可於盤點總覽頁面中的「資產所在位置」欄位查看變更。

![圖五 - 完成畫面](../../images/cf427f2ecbb459eb.png)
{% endstep %}
{% endstepper %}

### 03 - 3｜篩選資產

當資產資料較多時，您可使用篩選器，選擇所在位置及分類，快速篩選並找到欲查詢的資產項目。

![圖六 - 資產篩選說明 (一)](../../images/64dd28f5a1584d87.png)

如圖七所示，點選分類欄位即可開啟分類篩選表，您可勾選多個資產分類，以進行條件篩選。

![圖七 - 資產篩選說明 (二)](../../images/583a898ccf20eaeb.png)

輸入篩選條件並確認無誤後，點選「![](../../images/40d04d101ec9d1ce.png)」即可查找相符的配件資料，實例畫面如圖八所示。
